## 第一章 SOR 模型

### 1.1 基本定义

大观学派将人类一切心理活动简化为三个元素：

| 元素 | 全称 | 含义 | 属性 |
|------|------|------|------|
| **S** | Stimulus（刺激） | 客观存在的外部事件 | 可正、可负、可零 |
| **O** | Object（认知主体） | 个体的认知/诠释系统 | 决定性的转换器 |
| **R** | Response（反应） | 经由认知加工后的反应 | 可正、可负、可零 |

### 1.2 核心原理

**S 的属性不由事件本身决定，而由它穿过 O 之后 R 的表现来决定。**

- 经典案例：「好心好意变成驴肝肺」—— 你释放正 S，对方回以负 R。问题在对方的 O（负向认知状态）。
- 客观而言，世上发生的所有事情，就是其该发生的。既然是该发生的，就无所谓正负零。

### 1.3 治疗意义

- **不能只看 S 或 R，必须看到中间的 O**。
- 处于负 O 状态下的人，无论正 S 还是负 S 穿过，出来全是负 R。
- 治疗的核心不是改变 S，而是干预 O（认知系统）或直接处理 R（行为反应）。
- **自查应用**：为什么你对家人说好话、做好事，对方反应却不好？不是事件问题，是对方 O 状态的问题。

### 1.4 SOS 热线基本信念

能打电话进来本身，说明案主已具备一定的能力和动力。干预者的任务是**激发其潜能**，而非替代其功能。

## 第二章 六大心理连带关系与幸福盘/痛苦盘

### 2.1 六大心理连带关系

这是大观学派对「人何以成为人」的核心框架。贯穿「我」的生命有六条钢索：

| 连带关系 | 定义 | 说明 |
|----------|------|------|
| **生命** | 生死相连，不能独生不能独死 | 最根本的连带 |
| **生理** | 与自己的身体/性伴侣的关系 | 身体的健康与存在感 |
| **心理** | 想起对方心里就甜甜的幸福感 | 情感的归属 |
| **生活** | 日常共处互动的习惯模式 | 日常的扎根 |
| **经济** | 给/花用钱的关系 | 物质的安全 |
| **社会** | 社会角色对应关系（夫-妻、父-子等） | 角色的确认 |

### 2.2 幸福盘

将令人幸福的人、事、物按 **+1 到 +5** 的强度填入六个区，得到六种幸福指数。

**幸福盘四大危机**：
1. 六大连带关系五个区块都没人
2. 五个区块都是同一个人
3. 五个区块的人都不是家人
4. 觉察后不主动经营，自暴自弃

### 2.3 痛苦盘

将令人痛苦的人、事、物按 **-1 到 -5** 的强度填入六个区，得到六种创伤指数。

**痛苦盘六大危机**：
1. 每一种连带关系都受创伤
2. 创伤源都是同一个人/事/理/境
3. 创伤源都是亲人
4. 震央在某一关系，但灾区扩大到多种关系
5. 创伤源不在幸福盘或分数很低
6. 觉察后不主动改变

### 2.4 治疗应用

- 同一事物可同时在多个维度产生正分和负分。例如：儿子的出生（生命+5），但成绩不好（社会-3）。
- 拉力 = 幸福指数，推力 = 痛苦指数。
- **自杀发生在拉力小于推力时**。关键不在于推力多大，而在于有没有拉力。

## 第三章 人类创伤反应发展模型

### 3.1 模型概览

该模型是林昆辉对自杀预防最核心的理论贡献。它将从创伤事件到自杀的全过程分为 **A 到 J 十个阶段**。

**核心原则**：
- A→J 具有序阶性（层层加深）
- H 阶段（自杀）非常特殊：**A→G 任一阶段都可能直接跳到 H**
- 每一阶段的反应（R）本身可成为下一阶段的刺激（S）
- 治疗就是在这条下坠的路径上找到插入点，逆转方向

### 3.2 各阶段详解

```
A（创伤事件）→ B（创伤心理）→ C（创伤生理）→ D（偏差行为）
→ E（人格违常）→ F（精神官能症）→ G（精神分裂症）
→ H（自杀）→ I（自残/自虐）→ J（意外死亡）
```

#### A 阶段：创伤事件的反应与刺激

**创伤事件的三大种类**：
1. 不可能解决的事件
2. 可以解决但资源不足且资源系统已封闭
3. 碰都不能去碰的事件

**创伤事件后的五段序阶反应**：

| 阶段 | 表现 | 介入策略 |
|------|------|----------|
| 否认期 | 「哪有可能？」 | 只能催化渡过速度 |
| 自怜期 | 「为什么是我？」 | 只能催化渡过速度 |
| 羔羊期 | 「谁害的？」 | 只能催化渡过速度 |
| **求助期** | **「怎么办？」** | **此时才能提供援助** |
| 创伤期 | 「受不了，不想活了」 | 需转介心理治疗 |

> **关键**：前三个阶段（否认、自怜、羔羊）只能陪伴、催化渡过速度，不能给建议。只有到求助期才能提供实质性援助。

#### B 阶段：创伤心理的反应与刺激

呈现「极端」或「异常」的动机与情绪：
- **极端**：长时间停留在负向状态，但尚可自我控制
- **异常**：越来越长、越来越不能控制、失去对象性、变成强迫性

**核心公式**：
```
心理 = 动机（起心动念）+ 情绪（感受）
创伤心理 = 负向动机 + 负向情绪（二者必有其一为负向）
```

**治疗方向**：向前看——新的体验产生新的动机和新的情绪，可能替换旧的动机和情绪。

#### C 阶段：创伤生理的反应与刺激

| 成人反应 | 儿童反应 |
|----------|----------|
| 肩颈背酸痛 | 不动/过动 |
| 全身无力/僵直 | 不食/暴食 |
| 胸闷心悸 | 不语/多语 |
| 没胃口 | 失眠/嗜睡 |
| 失眠/嗜睡 | 尿失禁 |
| 性功能障碍 | — |

**创伤生理三阶段**：
1. 初期：未固化
2. 中期：特定生理反应固化，且肌肉筋骨「硬酸痛」**不会随心理好转立刻消失**
3. 末期：脱离因果关联，变成器质性生理疾病

#### D 阶段：偏差行为的反应与刺激

五种偏差行为：违反礼仪 → 违反常规 → 违反常理 → 违反人情 → 违反法律

**介入技术**——「指正」而非「指错」：
- 绝口不问是非真假对错
- 直接表达相信对方知道该怎么做
- 相信他有不得已的苦衷
- 帮他拿梯子「下错的台」同时「上对的台」

**三大禁忌**：
1. 不要「指错」
2. 不要做负向的「他人预言」
3. 不要做「人身攻击」

#### E 阶段：人格违常

与 D 阶段的关键区别：D 阶段自知是非对错；**E 阶段不认为自己错**——「只有需要，没有是非对错」。

五种类型：类分裂型、反社会型、边缘型、做作型、依赖型

#### F 阶段：精神官能症

- 忧郁症（轻/中/重）：生理症候群 + 心理症候群（什么都不、作息混乱、负向情绪、想死）
- 躁郁症：忧郁 + 狂躁期交替
- 焦虑症：无法控制的担忧与恐惧

> **重要辨析**：关于「重度抑郁症不会自杀」的说法存在争议。课件中标注了对此的质疑。临床上应独立判断。

#### G 阶段：精神分裂症

失去知觉与表出能力，动机与情绪关联分裂。以精神药物治疗为主。

#### H 阶段：自杀

自杀方程式 S-R 模型：

```
S1（事件/心理/行为的冲突困局）+ S2（费尽一切力量都解决不了）
→ R3（自杀动机）
→ S4（撞针事件）
→ R5（自杀行为）/ R6（自残行为）/ R7（自虐行为）
```

**四大自杀行为分类**：
1. **意外性自杀**：操作前置事件将自己置入险境，然后「意外」死亡
2. **动机性自杀**：用肉体痛苦取代心灵痛苦
3. **目的性自杀**：把自杀当作生活目的
4. **病发性自杀**：病情发作丧失控制能力（以忧郁症最多）

#### I 阶段：自残/自虐

- **自残**：用濒临致死的行为对抗「想死」的内在威胁，用肉体痛苦替代心理痛苦
- **自虐**：用凌迟肉体的行为惩罚自己「想死」，获得「主宰感」

> **反直觉理解**：自残是「自救」——当事人在用肉体疼痛解除心理疼痛。批评只会让他觉得你更不理解他。

#### J 阶段：意外死亡

前置事件几乎都是当事人「故意」或「刻意」或「坚持」去做的。放任事件逼成尖锐对立、高度冲突，然后「意外」死亡。

## 第四章 生命危机等级表

### 4.1 四级七等 26 级总览

这是大观学派危机干预最核心的诊断工具。从 A 到 Z，26 个等级，对应 7 个等别、4 个危机层级。

### 4.2 详细分级

#### 第一等：轻度危机——生活事件失控（A-G 级）

**核心认知**：「千错万错都是别人的错」——没有能力控制环境，归罪于他人、他事、他物。

| 等级 | 名称 | 说明 |
|------|------|------|
| A | 事理不平 | 外在事件与自己内在「理」不匹配，一切问题的起点 |
| B | 人际困扰 | — |
| C | 无解决方案 | — |
| D | 忧思苦恼 | — |
| E | 痛苦绝望 | — |
| F | 冷漠 | 「就这样吧，该咋样就咋样」；视而不见，听而不闻 |
| G | 精神困扰 | 心理和生理都不好（如睡不好导致控制不好情绪） |

**脱口语**：「不想活了」

> 轻度危机还不会威胁生命，应找亲友谈谈。

#### 第二等：轻度危机——人际冲突挫败（H-J 级）

| 等级 | 名称 | 说明 |
|------|------|------|
| H | 对环境负向观 | 泛化或灾难化，从环境中抽离，批评否定世界 |
| I | 对他人负向观 | 批评排斥他人 |
| J | 对自己负向观 | （过去、现在、未来）批评否定自己 |

#### 第三等：中度危机——异常动机困扰（K-O 级）

**核心认知**：「千错万错都是自己的错」——特定事件或他人已消失，转而沉沦于自身的负向动机或负向情绪。

| 等级 | 名称 | 说明 |
|------|------|------|
| K | 深度焦虑、恐慌、畏惧、强迫 | — |
| L | 对自己/他人/社会的敌意 | 受害者心态、报复心、想攻击 |
| M | 对生命的敌意 | 「为何没经过我同意就把我生下来」 |
| N | 家庭责任瓦解 | 「家里没我也挺好」 |
| O | 生存信念瓦解 | 不能确认下一秒是否活着 |

> O 级的关键表现：「活一天是一天」的亡命徒心态。「明天、坚持、希望、努力」都失去意义。

#### 第四等：中度危机——异常情绪困扰（P-R 级）

| 等级 | 名称 | 说明 |
|------|------|------|
| P | 重郁、大哭或狂躁 | 无前置条件 |
| Q1 | 不快乐 | 「活不下去了」（被动型） |
| Q2 | 活不下去 | — |
| R1 | 太痛苦了 | — |
| R2 | 死了算了 | 「常常想去死」（主动型） |

**脱口语**：「死了算了」

> 关键辨析：不快乐 ≠ 太痛苦。不快乐常意味着「活不下去」（被动），太痛苦常意味着「想去死」（主动）。

**区分轻度和中度危机的关键**：有没有**前置事件**。
- 有具体事件触发 → 常态反应，即使表现严重仍是轻度
- 没有发生任何事但天天想死、天天哭 → 异常，已泛化，进入中度

#### 第五等：重度危机——已启动自杀方程式（S-U 级）

**核心认知**：不再攻击任何人事物，觉悟「没有能力继续活下去」。

| 等级 | 名称 | 说明 |
|------|------|------|
| S1/S2/S3 | 自杀动机（过去/现在/未来） | 「因为 S 我才自杀」（有前置事件） |
| T | 自杀意念 | 「只要是自杀就 OK」（无前置事件） |
| U | 目的性自杀 | 「只有我自杀了，他们才会……」 |

> 付老师类比：自杀动机像「因为家长说考好去旅游所以学习」；自杀意念像「不知道学习有什么用但就是要学习」；目的性自杀像「我学习是为了建设祖国」——工具化程度越深越危险。

#### 第六等：重度危机——准备进入自杀程序（V-W 级）

| 等级 | 名称 | 说明 |
|------|------|------|
| V1 | 自伤（自虐、自残、自杀）经验 | — |
| V2 | 无法制止自杀的处境 | — |
| W1 | 自杀安排 | — |
| W2 | 临终安排 | — |

**脱口语**：「我要自杀」

> 需立即找人陪伴，避免独处。

#### 第七等：急迫危机——已进入自杀程序（X-Z 级）

**核心认知**：觉悟自己没有能力救活自己，出现立刻寻死的语言或行为。

| 等级 | 名称 | 说明 |
|------|------|------|
| X | 立刻去死 | 现在就自杀的冲动或行为 |
| Y | 病发失控 | 共病失控症状（心率加快、胸闷等） |
| Z | 自杀行为（前/中/后） | 正在执行自杀 |

**脱口语**：「非死不可」

> 生命有立即性危险，必须「救下来才能挂断」。

### 4.3 危机等级诊断方法

#### 四句关键判断句

只能判断「是否在表上」（有无生命危机），**不能直接判断级别**：

1. 「不想活了」
2. 「死了算了」
3. 「我要自杀」
4. 「非死不可」

#### 级别判断的真正依据

**不是听他说什么话，而是听「话前面的内容」**：

| 他说的是…… | 危机等级 |
|-------------|----------|
| 事件（S1 = 没有能力控制事件） | 轻度 |
| 自己控制不了自己（S2 = 没有能力控制自己） | 中度 |
| 启动自杀方程（S3 = 没有能力救活自己） | 重度 |
| 行为已启动（S4 = 没有能力阻止自己） | 急迫 |

> **反例**：他说「不想活了」，但人站在楼顶上马上就要跳——你不能因为他说「不想活了」就评轻度。他是 S4：没有能力阻止自己，是急迫危机。

#### 逐字稿对应法

将来电者的话对应成逐字稿，每说一句就对照生命危机等级表做勾选判断。

#### 动态评估原则

- 危机等级会跳来跳去，不是线性固定的
- **以连续被勾选的最后落脚点（最低最严重的等级）为标准**
- 否则容易高估或低估

> **案例**：来电者说「我以前自杀过」（V1，重度），但当下说的是「爸爸打我凭什么」（A，事理不平，轻度）→ **当下判读是轻度**。不能因为有自杀史就拔高。

#### 「在河边/在天台」≠ 急迫危机

不要仅凭地理位置高危就拔高等级。必须结合言语内容判断。案例：四年级孩子说在十楼窗台上，但谈话内容全是事理不平（父母偏心），无任何关键句 → 连危机个案都不是，是咨询个案。

### 4.4 危机解除的阳性指标（可观察的降级信号）

1. 小声变大声
2. 被动发言转主动发言
3. 短句变长句
4. 从「怎么死」转向「怎么活」
5. 从「自己」转向「他人或物品」
6. 谈话焦点从「人」转到「事」，从「我」转到「我的」
7. 谈话主题从低等别向上移动（如从 5/6 等升至 3/4/2/1 等）

## 第五章 接访流程与时间管理

### 5.1 整体流程

```
前 15 分钟           →        后续
事件陈述阶段                    干预阶段
├─ 建立有疗效的咨访关系            ├─ 根据危机等级选择技术
├─ 做生命危机等级诊断              ├─ 轻中度 → A-F 六阶段
└─ 不收集信息（不打断）            ├─ 重度 → 搬楼层技术
                                └─ 急迫 → NOPQ 技术
```

### 5.2 事件陈述完成的三个标志

1. 案主直接问「怎么办」
2. 案主开始重复讲述
3. 即使 15 分钟还没到，上述两个标志出现即证明陈述可结束

### 5.3 时间管理标准

| 危机等级 | 范围 | 干预类型 | 时间限制 |
|----------|------|----------|----------|
| 轻度 + 中度 | A-P | 心理咨询 | **30 分钟内结案** |
| 重度 | Q-W | 降低危机等级 | **60 分钟内结案** |
| 急迫 | X-Z | 救下来才能挂断 | **不限时** |

### 5.4 信息收集时机

- 前 15 分钟：**禁止收集个人信息**（年龄、身份等）
- 只能在「谈事」的空档收集
- 案主在谈生死喜怒哀乐时不能打断收集信息
- 目的：保护治疗联盟的建立

## 第六章 六变三托

### 6.1 定义

六变三托是识别自杀行为征兆的核心判断工具。来自人类创伤反应发展模型中 R7-R9 级别的关键预警信号。

### 6.2 六变

| 序号 | 名称 | 具体表现 |
|------|------|----------|

| 2 | **行为大变** | 不按规律习惯作息，该干嘛不干嘛，不该干嘛去干嘛 |

| 4 | **语言大变** | 突然谈论或书写与死亡相关的话题 |
| 5 | **身体大变** | 突然出现与生理疾病不符的身体症状，或突然外表大变 |
| 6 | **环境大变** | 突然对环境产生巨大变化（搬家、转学、换工作等） |

| 3 | **托物** | 突然打包身边重要文物、玩物或宠物，要求或委托代为照顾或保管 |

### 6.4 临床注意

## 第七章 同步同理心技术

### 7.1 定义

同步同理心是前 15 分钟与案主建立有疗效咨访关系的核心技术。

### 7.2 操作要点

| 要素 | 方法 | 说明 |
|------|------|------|
| **声音三同步** | 音量、语速、情感跟随案主 | 案主小声你小声，案主快你快 |
| **感叹词** | 嗯、哦、啊、唉 | 同一个感叹词不能连用（不能说「嗯嗯嗯嗯」） |
| **连接词** | 后来呢？接下来呢？ | 推动陈述 |
| **鹦鹉句** | 重复案主的关键词句 | 「他说你不行？」 |
| **话要比案主少** | 不插话，不问「为什么」，不问「凭什么」 | 不能长篇大论 |

### 7.3 同步同理心五大效标

| 案主 | 接线员 |
|------|--------|
| 我的事件 → 充分表出 | 我真心聆听 → 同步表出 |
| 我的情绪 → 充分宣泄 | 我帮忙释放 → 全部承受 |
| 我的动机 → 被了解 | 我充分了解 → 绝对相信 |
| 我的行为（反应）→ 被允许 | 我允许认同 → 绝对肯定 |
| 不幸的我（人）→ 被接受 | 我完全接受 → 价值见证 |

### 7.4 最终目标

建立「**Cl（案主）— CO（助人者）— 世界**」新的连带关系。

### 7.5 第一句话的禁忌

**绝对不能问**「我能怎么帮助您」「请您说说您这次接线有什么感觉」——这来自其他流派。

**正确开场**：「感恩您的信任。」

**核心心态**：把来电者当作「生死之交」，带着感恩的心接听。

## 第十章 A-F 六阶段求助等级发展模型

### 10.1 模型概览

这是轻中度危机干预的核心操作流程。**原则：不一定全部走完 A 到 F，根据案主需求灵活运用。到哪算哪。**

### 10.2 各阶段详解

#### A — 事件（表出阶段）

| 项目 | 内容 |
|------|------|
| 案主需要 | 充分表出 |
| 接线员任务 | 不插嘴、不问缘由、不否定 |
| 技术 | 感叹词 + 连接词（「后来呢」「接下来呢」）+ 鹦鹉句 |

#### B — 情绪（宣泄阶段）

| 项目 | 内容 |
|------|------|
| 案主需要 | 充分宣泄 |
| 接线员任务 | 完全承接 |
| 技术 | **人称技术** |

**人称技术五级用法**：

| 人称 | 示例 | 效果 |
|------|------|------|
| 无人称 | 「能不生气吗？」 | 普遍化 |
| 第一人称单数 | 「我听了都生气」 | 共情 |
| 第一人称复数 | 「我们怎么可能不生气」 | 联合 |
| 第二人称 | 「难怪你这么生气」 | 确认感受 |
| 第三人称 | 「他太过分了」 | 外化 |

> **角色扮演技术**：当案主需要时，接线员扮演那个对不起她的人，说「对不起，我错了」——这是她在生活中从来没听到过的话。

#### C — 动机（肯定阶段）

| 项目 | 内容 |
|------|------|
| 案主需要 | 被肯定 |
| 接线员任务 | 肯定其动机的合理性，不评判对错 |
| 示例 | 「你当然会这样想啊，怎么可能不这样想。你这样想是合理的」 |

#### D — 行为/反应（赦免阶段）

| 项目 | 内容 |
|------|------|
| 案主需要 | 被赦免 |
| 接线员任务 | 使用「不得不」的语境 |
| 示例 | 「那种情况下，你也是不得不这样做」 |

#### E — 不幸的人（价值见证阶段）

| 项目 | 内容 |
|------|------|
| 案主需要 | 被见证 |
| 接线员任务 | 见证其价值与不易 |
| 技术名称 | 「给她戴上痛苦的冠冕」 |
| 示例 | 「这个家老有你，只有你能挺得住。你真是了不起的妈妈」 |

> **赞美时机**：一定要在案主自己展现出积极面时再赞美。不能在她还在抱怨时就夸。她说「可我还是得管孩子」→ 「你真是了不起的妈妈」。她说「吃中药不管用还得管孩子」→ 这时候夸就错了。

#### F — 我的希望（赋能阶段）

| 项目 | 内容 |
|------|------|
| 案主需要 | 被赋能 |
| 接线员任务 | 激发案主自己的办法 |
| 核心原则 | **绝不低估来访者**。激发案主自己想到的办法才是他自己的方法 |

**递进式提问序列**：
1. 「你喜欢现在的自己吗？」
2. 「这样的日子要继续吗？」
3. 「真的要改变吗？」
4. 「非改不可吗？」
5. 「对你来说最重要的是什么？」
6. 「为了孩子愿意做点啥？」
7. 「要不要先照顾好自己呢？」

> 整个过程不给建议——「人家能想到的办法人家都知道。」启发案主自己想出来的答案才是他要的。

## 第十一章 语言六大因子与搬楼层技术（重度危机干预）

### 11.1 SOR + PNF 语言六大因子

| 因子 | 全称 | 所属空间 | 含义 |
|------|------|----------|------|
| S | Stimulus | 心理空间 | 刺激/事件 |
| O | Object | 心理空间 | 认知/诠释 |
| R | Responds | 心理空间 | 反应/回应 |
| P | Past | 心理时间 | 过去 |
| N | Now | 心理时间 | 现在 |
| F | Future | 心理时间 | 未来 |

### 11.2 搬楼层技术（核心实战技术）

**原理**：沿着生命危机等级表，从高往低带。

```
6 楼（自杀计划 W）→ 5 楼（想自杀 S/T/U）
→ 4 楼（太痛苦了/死了算了 Q1-R2）→ 3 楼（不快乐/活不下去 Q-P）
```

每层「打扫房间」（用 SOR/PNF 问句清理困扰），然后再「打包换楼层」（用打包问句把案主带到下一个等级）。

### 11.3 重度危机者特征

- 一般是 O/SO 语言占主导（沉浸在自我诠释中）
- 干预方向：进入他的语言系统 → 增加 SR 语言（具有治愈性的对话）→ 用打包技术换楼层

### 11.4 问句序列

#### 第一轮：探索自杀意念历程（从过去→现在）

| 序号 | 问句 | 语言因子 | 目的 |
|------|------|----------|------|
| 1 | 「那以前有过这想法吗？」 | PO | 拉出时间线 |
| 2 | 「第一次有这个自杀的想法是什么时候？」 | PS | 定位于过去 |
| 3 | 「那时候遇到什么事儿让你有这个想法？」 | PSO | 探索事件+认知 |
| 4 | 「那个念头出现的时候，你有没有害怕呀？」 | PO | 探索身心状态 |
| 5 | 「随后又做了什么呢？」 | PR | 探索行为 |
| 6 | 「那你有跟爸妈说吗？」 | PR | 探索支持系统 |
| 7 | 「爸爸妈妈的反应是什么？」 | — | 探索他人反应 |
| 8 | 「那爸爸妈妈这样说，你感觉怎么样呢？」 | PSO | 探索感受 |
| 9 | 「那时候有这想法时，如何不去自杀的？」 | — | **找过去的资源** |
| 10 | 「那你成功阻止自杀后，人生有没有什么变化？」 | — | 确认有活过来的经验 |
| 11 | 「那现在发生了什么事呢？让你冒出这个自杀的念头」 | NSR | 切入当下 |
| 12 | 「那现在的你如何看待过去引起你自杀的事情呢？」 | NOS | 从现在的视角看过去 |

#### 第二轮：打包换楼层

| 序号 | 问句 | 技术要点 |
|------|------|----------|
| 1 | 「从过去到现在有好几次要自杀，这样过日子，你有办法不去死吗？」 | 打包过去经验 |
| 2 | 「你经历过这么多，太痛苦了，你会不会觉得……」 | **未完成句**——让案主自己说出「死了算了」 |
| 3 | 「那你上次觉得『死了算了』又活了过来，那这次有可能再活过来吗？」 | **双向问句**——同时包含「活」和「死」 |
| 4 | 「怎么都是要死，你还快乐得起来吗？你一定很……」 | 未完成句 |
| 5 | 「这样的不快乐会不会哭，会不会叫？」 | 探索情绪 |
| 6 | 「你会想你的家人吗？」 | 重建连带 |
| 7 | 「你的家人会想你吗？」 | 重建连带 |
| 8 | 「不重要多久了？」 | 量化时间 |

### 11.5 关键技术要点

**未完成问句**：
- 「你会不会觉得不如死了……」——停顿，让案主自己说完
- 「不如死了算了」由案主自己说出来，比接线员说出来强力得多
- 如果案主不说，接线员再说「不如死了」

**双向问句**：
- 「你觉得这次有可能再活回来吗？」——同时包含「活」和「死」两种可能
- 「怎么说都是要死，你快乐的起来吗？」——痛苦和快乐对比

**面对自杀念头时**：
- 立即说「想想就好，想想就好，不做」
- 把「想」和「做」分开
- 不要问他为什么想自杀，不要问原因

**找过去的资源**：
- 核心问句：「那时你是如何让自己没有去自杀的？」
- 目的是挖掘案主已有的生存策略

### 11.6 搬楼层三大原则

1. **不能回避谈生死，不能害怕谈生死**——要与案主谈论生与死
2. **核心是组合拳 + 情感**
3. **当案主跟得上谈论较轻等级时，即代表危机等级已减轻一等**

## 第十二章 NOPQ 急迫危机干预技术

### 12.1 模型概览

这是针对第七等（急迫危机 X-Z）的专项干预技术。整体流程：

```
N（承诺救援 + 激发求生意志 + 移动位置 + 询问地址 + 保持通话）
→ 成功 → 做生命等级评估 → 打包往 W → 降低危机等级 → 结案
→ 失败 → O（转接公安局）
→ P（禁止自杀行为）
→ Q（报警）
```

### 12.2 N 语言（N1-N5）

#### N1：承诺救援

> 「我要救你，我一定要救你。请你一定要帮我，让我来救你。你一定要给我机会救你，你要帮我，请你一定要帮我。」

- 第一人称 + 第二人称关系句
- 表达坚决的救助意愿

#### N2：激发求生意志

> 「一定要活下去，我们都要活下去，你一定要活下去。」

- **三段式重音变化**：重音位置每句不同
- **坚决不去帮他做决定**——不是问「你为什么要自杀」
- 添加情感温度，不能机械念

#### N3：要求移动位置

> 「手抓好，移动一下左脚。左脚还是右脚？左脚在里面还是右脚在外面？好，来移动一下。休息一下，靠一下。我一定会救你。」

- N1 和 N2 的真正目的是**制约**案主——在小的事上顺从后，才能要求他做关键行为
- 这是整个 N 语言的核心：**移动到安全位置**

#### N4：询问地址

> 「你在哪里？」

#### N5：保持通话 + 生命等级评估

询问 X/Y/Z：
- **X**：「你现在有没有可能停下来？」——冲动控制
- **Y**：评估共病失控症状（心率加快、胸闷等生理信号）
- **Z**：询问自杀前、中、后的状态

### 12.3 P 语言（P1-P3）

#### P1：禁止自杀行为

转移自杀行为。

#### P2：「改签」技术

不是劝阻自杀，而是改变自杀方式/地点/时间：

| 类型 | 话术示例 |
|------|----------|
| 换地点 | 「这个地方不好，毛毛的，拜托拜托换个地方」 |
| 换方式（跳楼） | 「血肉模糊，脑浆迸裂多吓人」 |
| 换方式（喝药） | 「肠子多难受多烧得慌」 |
| 换方式（跳河） | 「尸体一泡多恐怖，鱼会吃你」 |
| 换时间 | 「为了我，为了陌生的我，今天不要死。我们生命就连在一起了」 |

#### P3：最后的协议

> 「至少答应我，今天要活着可以吗？」
> 「我活着，你也得活着，我不许你死。」
> 「不管别人在不在乎你，我要告诉你，我在乎你。不要你死。」
> 「为了陌生的我，你也要活下来。」

- 必须果断、坚决、斩钉截铁
- 必须有情感——没有情感就是冷冰冰的文字

### 12.4 O 语言：转接公安

N 语言一旦失败，转接 O（公安局/警察）。

### 12.5 Q 语言：报警程序

**关键规则**：
- **接线员无权直接报警**——必须联系当日线上督导，由督导决定
- 单人值班时：必须立即寻求督导协助
- **严禁承诺不报警或不通告家人**——这不是接线员能做出的承诺
- 报警话术含工号、地点和具体协助请求
- 报警后必须回拨确认追踪结果

**案主拒绝报警时**：
> 「我来就没人接线了，我会让我们相信的人去」

### 12.6 N 语言重音训练

| 规则 | 说明 |
|------|------|
| 五个字一个重音 | 均匀分布强调 |
| 最后一个字要转换音色 | 避免习惯性尾音 |
| 复合句重音不在同一位置 | 每句重音位置不同 |
| 前后两句要转换音色 | 避免单调 |

### 12.7 接线要求

- 态度要坚决，不能犹豫
- **不要使用敬语**
- 语气果断斩钉截铁

### 12.8 回拨机制

| 个案级别 | 回拨频率 | 时间窗口 |
|----------|----------|----------|
| 7 级（急迫） | 每 4 小时 | 上午 10:00 - 晚上 10:00 |
| 6 级及以下 | 由督导决定 | — |

- 7 级个案当下断线：**立即回拨**，不通则报警
- 危机降级后才可调整回拨频率

## 第十三章 治疗性会谈五阶段模型

这是针对自杀（未遂）个案的完整治疗操作模型。

### 第一阶段：一般关系之建立（新的、信任的连带关系）

| 要做 | 不要做 |
|------|--------|
| 全时陪伴 | 不要抱怨、不要质问 |
| 生活起居陪伴，不厌烦 | — |
| 生活照顾 | 不要劝善、不要规过 |
| 和颜悦色 | 不要哀求 |
| 温言软语 | 不要责骂 |
| 当事人主动诉说时：不插嘴、不问话 | — |
| 当事人长时间不说话：不问原因，生活杂谈，增加互动量 | — |

**目标**：重新挂上六大连带关系。

### 第二阶段：治疗关系之建立

1. 见证六大连带关系已破灭
2. 表出模型之抉择见证
3. 负向自我状态的觉察
4. 负向态度的觉察
5. 企求解脱的动机意念的引发与抉择
6. 新的治疗性连带关系的抉择

### 第三阶段：同步同理心五大阶段

使用同步同理心技术（参见第七章）。

### 第四阶段：封印旧我、创建新我

**摸彩箱与彩球隐喻**：
- 人生像一个摸彩箱，别人丢入的黑球（创伤）不会消失
- 但你可以丢入大量红球（积极经验），让摸到黑球的概率变小
- 球已经满了？**封印旧箱，换新箱**

**封印的程序**：
```
接受（已成为事实）
→ 原谅（自己无力回天）
→ 不甘心（封印的动力与抉择的勇气）
→ 抉择
→ 新生
```

**关键**：
- 不响应负向循环语言（「为什么我还活着」「我不要活了」等）
- 肯定过去连带关系的经验价值：当事人的所有反应和抉择都是合理的、正常的、必要的、重要的、有价值的

### 第五阶段：发展新我与六大连带关系价值工程

1. 新旧价值定位与排序（幸福盘与痛苦盘的比较分析）
2. 动机管理：切断负向循环，主动创造与维护高价值「动机-行为-情绪」
3. 低价值「动机-行为-情绪」的主动切除与封印
4. 自我管理：处理善后事件，制定程序
5. 生涯规划作业，制定生活计划表
6. 主动营建新的六大连带关系
7. 家族治疗：调适家庭角色行为与关系

## 第十四章 接线员自我保护

### 14.1 入戏与出戏

| 概念 | 含义 | 操作 |
|------|------|------|
| 入戏 | 通话期间完全扮演接线员角色 | 放下个人身份 |
| 出戏 | 通话后的自我保护 | 防止替代性创伤 |

### 14.2 高危来电后的处理

- **必须主动联系督导**进行情绪复盘
- 防止耗竭
- 支持结构：背后有 8 名以上线上督导
- 杜老师分享：第一次急迫危机个案 2.5 小时，「浑身是冷的」
- 林昆辉：**「保护接线员比保护案主还重要」**

### 14.3 报警权限

- 接线员**没有权利**自行报警
- 必须找当天线上督导请示
- **严禁承诺不报警或不通告家人**

## 第十五章 常见错误与禁忌

### 15.1 六大禁忌话术

| 禁忌类型 | 错误示例 | 问题 |
|----------|----------|------|
| **劝善** | 「生命是宝贵的」「给自己一个机会」 | 否定案主的痛苦体验 |
| **规过** | 「你对得起父母吗？」「你怎么可以这样想」| 加重罪咎感，加速自杀 |
| **哀泣** | — | 情绪卷入过度 |
| **责骂** | — | 破坏治疗关系 |
| **质问** | 「你凭什么……」 | 让案主更防御 |
| **抱怨** | — | 丧失中立立场 |

### 15.2 其他禁忌

- **严禁自我暴露**：不能说「我以前也这样」「我的孩子也怎样怎样」
- **忌用「聊」「说」**：「我们聊聊」「我们说说」——这是闲聊，不是专业干预
- **同一个感叹词不能连用**：「嗯嗯嗯嗯」让案主感觉你在敷衍
- **不能提前赞美**：赞美必须在 E 阶段（价值见证）且案主已展现积极面时
- **不能给建议**：激发案主自己想出来的才是他的方法
- **前 15 分钟不能收集信息**：打断会破坏联盟建立
- **不能因为地理位置高危就拔高危机等级**：必须结合言语内容
- **如果案主痛苦源是家人，不能说「为了家人」**：适得其反

### 15.3 新手常见错误清单

1. 接电话第一句就问「我能怎么帮助您」
2. 在情绪宣泄阶段过早赞美
3. 劝善、劝归、道德说教
4. 自我暴露
5. 使用「说」「聊」等闲聊词汇
6. 感叹词连用或情感错位
7. 「好心好意成驴肝肺」——送正 S，收获负 R，不理解为什么（忽略了对方的 O 状态）
8. 拉力变推力的识别——长期互动中，曾经的拉力可能变成推力
