# 生命危机评估使用说明

## 接访记录中的生命危机评估

在接访记录表单中，已经集成了大观学派的生命危机等级评估系统。

### 评估体系概览

**26个级别（A-Z）** 分为 **7个等别** 和 **4个危机层级**：

| 危机层级 | 等级范围 | 干预时长 | 干预技术 |
|---------|---------|---------|---------|
| 🟢 轻度危机 | A-J（第一、二等） | 30分钟结案 | A-F六阶段求助等级技术 |
| 🟡 中度危机 | K-R（第三、四等） | 30分钟结案 | A-F六阶段求助等级技术 |
| 🔴 重度危机 | S-W（第五、六等） | 60分钟结案 | 搬楼层技术 |
| ⛔ 急迫危机 | X-Z（第七等） | 不限时！ | NOPQ技术 + 立即报警 |

### 四大脱口语判断

1. **"不想活了"** → 轻度危机（A-J）
2. **"死了算了"** → 中度危机（K-R）
3. **"我要自杀"** → 重度危机（S-W）
4. **"非死不可"** → 急迫危机（X-Z）

### 核心判断依据

**不是听来访者说什么话，而是听"话前面的内容"：**

| 来访者说的是... | 危机等级 |
|-------------|---------|
| 事件（S1 = 没有能力控制事件） | 轻度 |
| 自己控制不了自己（S2 = 没有能力控制自己） | 中度 |
| 启动自杀方程（S3 = 没有能力救活自己） | 重度 |
| 行为已启动（S4 = 没有能力阻止自己） | 急迫 |

### 在表单中如何使用

1. **选择危机等级**：从下拉菜单中选择A-Z中的具体等级
2. **查看自动显示的信息**：
   - 危机层级（轻度/中度/重度/急迫）
   - 危机等别（第一等到第七等）
   - 脱口语（来访者典型表述）
   - 核心认知（该等级的心理特征）
   - 干预策略（推荐的干预技术和时间）

3. **记录评估依据**：在"评估依据与观察"文本框中记录：
   - 来访者的关键对话
   - 情绪表现
   - 行为特征
   - 前置事件等

4. **勾选六变三托征兆**（如有）：
   - **六变**：性情大变、行为大变、财务大变、语言大变、身体大变、环境大变
   - **三托**：托人、托事、托物

### 关键原则

#### 1. 动态评估
- 危机等级会跳来跳去，不是线性固定的
- **以连续被勾选的最后落脚点（最低最严重的等级）为标准**

#### 2. 区分常态与异常
**轻度 vs 中度的关键区别：有没有前置事件**
- 有具体事件触发 → 常态反应，即使表现严重仍是轻度
- 没有发生任何事但天天想死、天天哭 → 异常，已泛化，进入中度

#### 3. 地理位置不等于危机等级
"在河边/在天台" ≠ 急迫危机
- 必须结合言语内容判断
- 例：孩子在十楼窗台，但谈话内容全是事理不平（父母偏心），无任何关键句 → 连危机个案都不是

#### 4. 历史不等于当下
来电者说"我以前自杀过"（V1，重度），但当下说的是"爸爸打我凭什么"（A，事理不平，轻度）→ **当下判读是轻度**

### 危机解除的阳性指标

观察到以下信号说明危机等级正在降低：

1. 小声变大声
2. 被动发言转主动发言
3. 短句变长句
4. 从"怎么死"转向"怎么活"
5. 从"自己"转向"他人或物品"
6. 谈话焦点从"人"转到"事"，从"我"转到"我的"
7. 谈话主题从低等别向上移动（如从5/6等升至3/4/2/1等）

### 特别注意

#### 急迫危机（X-Z）处理要点
- ⛔ **救下来才能挂断**，不限时
- 使用NOPQ技术
- **必须联系督导**，接线员无权直接报警
- **严禁承诺不报警或不通告家人**

#### 重度危机（V-W）处理要点
- 需立即找人陪伴，**避免独处**
- 脱口语："我要自杀"
- 60分钟内结案
