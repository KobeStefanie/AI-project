## 二、详细分类体系

### 1. 家庭关系

| 具体问题 | 常见表现 | 推荐治疗取向 |
|---------|---------|-------------|
| **亲子关系** | 亲子冲突、教育理念分歧、青春期叛逆、代际沟通障碍 | 家庭治疗、系统治疗、CBT |
| **夫妻关系** | 夫妻冲突、沟通不良、性生活不和谐、离婚危机 | 婚姻治疗、EFT情绪焦点治疗、系统治疗 |
| **出轨/婚外情** | 情感背叛、信任破裂、创伤反应、关系重建 | EFT、精神分析、存在主义 |
| **家暴** | 身体暴力、精神暴力、控制行为、创伤后应激 | 创伤知情治疗、EMDR、CBT |
| **赌博问题** | 赌博成瘾、家庭财务危机、信任破裂 | CBT、动机式访谈、家庭治疗 |
| **酗酒/药物滥用** | 物质成瘾、家庭破坏、共依存关系 | CBT、动机式访谈、12步疗法 |
| **婆媳关系** | 婆媳冲突、家庭界限不清、代际矛盾 | 家庭治疗、系统治疗 |
| **原生家庭创伤** | 童年创伤、父母关系创伤、家庭暴力史 | 精神分析、客体关系、EMDR |

### 2. 情感关系

| 具体问题 | 常见表现 | 推荐治疗取向 |
|---------|---------|-------------|
| **恋爱困扰** | 择偶困难、恋爱焦虑、依恋问题 | 依恋理论、CBT、人本主义 |
| **失恋/分手** | 分手创伤、哀悼反应、自我价值受损 | 悲伤辅导、存在主义、人本主义 |
| **暗恋/单恋** | 情感未回应、幻想过度、无法放下 | CBT、精神分析 |
| **社交恐惧** | 异性恐惧、亲密关系恐惧、回避型依恋 | CBT、暴露疗法、依恋治疗 |

### 3. 人际关系

| 具体问题 | 常见表现 | 推荐治疗取向 |
|---------|---------|-------------|
| **职场人际** | 同事冲突、上下级关系、职场霸凌 | CBT、人际关系治疗（IPT） |
| **朋友关系** | 友谊破裂、社交孤立、信任问题 | 人际关系治疗、团体治疗 |
| **社交焦虑** | 社交场合焦虑、回避社交、恐惧评价 | CBT、暴露疗法、接纳承诺疗法（ACT） |
| **边界问题** | 无法拒绝、过度付出、界限不清 | CBT、精神分析、自我心理学 |

### 4. 职业发展

| 具体问题 | 常见表现 | 推荐治疗取向 |
|---------|---------|-------------|
| **职业倦怠** | 工作疲惫、情感枯竭、成就感低 | CBT、正念减压（MBSR）、存在主义 |
| **职业规划** | 职业迷茫、生涯困惑、转型焦虑 | 生涯咨询、存在主义、人本主义 |
| **工作压力** | 工作负荷过大、时间管理、完美主义 | CBT、正念疗法、ACT |
| **失业困境** | 失业创伤、自我价值受损、经济压力 | 存在主义、CBT、正念疗法 |

### 5. 学业相关

| 具体问题 | 常见表现 | 推荐治疗取向 |
|---------|---------|-------------|
| **学习障碍** | 注意力不集中、学习效率低、考试焦虑 | CBT、行为疗法 |
| **考试焦虑** | 考前紧张、失眠、恐慌发作 | CBT、暴露疗法、正念疗法 |
| **校园霸凌** | 被霸凌创伤、社交退缩、自我价值受损 | 创伤治疗、CBT、EMDR |
| **学业压力** | 学业负担、父母期待、竞争压力 | CBT、家庭治疗、ACT |

### 6. 个人成长

| 具体问题 | 常见表现 | 推荐治疗取向 |
|---------|---------|-------------|
| **自我认同** | 身份困惑、自我价值低、角色冲突 | 存在主义、人本主义、叙事治疗 |
| **人生意义** | 存在空虚、意义感缺失、虚无感 | 存在主义、意义治疗、ACT |
| **自卑/羞耻** | 自我价值低、核心羞耻感、自我批判 | 精神分析、自我心理学、CBT |
| **完美主义** | 过度自我要求、害怕失败、拖延 | CBT、精神分析、ACT |
| **拖延症** | 拖延行为、时间管理困难、自责 | CBT、ACT、动机式访谈 |

### 7. 创伤相关

| 具体问题 | 常见表现 | 推荐治疗取向 |
|---------|---------|-------------|
| **性侵/性虐待** | 创伤后应激、信任破裂、解离症状 | 创伤知情治疗、EMDR、感知运动心理治疗 |
| **暴力目击** | 目击暴力、替代性创伤、恐惧反应 | EMDR、创伤治疗、CBT |
| **丧亲之痛** | 哀悼反应、悲伤困难、抑郁 | 悲伤辅导、存在主义、人本主义 |
| **重大事故** | 车祸、灾难、意外创伤 | EMDR、创伤治疗、CBT |
| **童年创伤** | 童年虐待、忽视、情感剥夺 | 精神分析、客体关系、图式治疗 |

### 8. 精神疾病（诊断分类）

#### 8.1 情绪障碍

| 具体诊断 | 常见症状 | 推荐治疗取向 |
|---------|---------|-------------|
| **抑郁症（轻度）** | 情绪低落、兴趣减退、疲倦 | CBT、人际关系治疗（IPT）、行为激活 |
| **抑郁症（中度）** | 上述症状加重、社会功能受损 | CBT + 药物、IPT、正念认知疗法（MBCT） |
| **抑郁症（重度）** | 严重抑郁、自杀意念、无法工作 | 药物治疗为主 + CBT、住院治疗 |
| **持续性抑郁障碍** | 长期低落情绪（2年+）、功能受损 | CBT、精神分析、图式治疗 |
| **双相情感障碍** | 躁狂/抑郁交替、情绪波动剧烈 | 药物治疗为主 + 家庭心理教育、CBT |

#### 8.2 焦虑障碍

| 具体诊断 | 常见症状 | 推荐治疗取向 |
|---------|---------|-------------|
| **广泛性焦虑障碍** | 持续担忧、紧张、身体症状 | CBT、正念疗法、ACT |
| **恐慌症** | 恐慌发作、回避行为、预期焦虑 | CBT、暴露疗法、呼吸训练 |
| **社交焦虑障碍** | 社交恐惧、回避社交、评价恐惧 | CBT、暴露疗法、团体治疗 |
| **特定恐惧症** | 特定物体/情境恐惧、回避行为 | 暴露疗法、系统脱敏、CBT |
| **创伤后应激障碍（PTSD）** | 闪回、回避、警觉过度、解离 | EMDR、创伤聚焦CBT、感知运动治疗 |

#### 8.3 强迫及相关障碍

| 具体诊断 | 常见症状 | 推荐治疗取向 |
|---------|---------|-------------|
| **强迫症（OCD）** | 强迫思维、强迫行为、焦虑 | 暴露与反应预防（ERP）、CBT、ACT |
| **躯体变形障碍** | 外貌过度关注、重复检查、回避 | CBT、ERP、正念疗法 |
| **囤积障碍** | 无法丢弃、过度收集、功能受损 | CBT、动机式访谈 |

#### 8.4 进食障碍

| 具体诊断 | 常见症状 | 推荐治疗取向 |
|---------|---------|-------------|
| **神经性厌食症** | 极度限制饮食、体重过轻、身体意象扭曲 | 家庭治疗（FBT）、CBT-E、住院治疗 |
| **神经性贪食症** | 暴食+清除行为、体重正常或偏高 | CBT-E、人际关系治疗、辩证行为疗法（DBT） |
| **暴食障碍** | 反复暴食、无清除行为、肥胖 | CBT、DBT、动机式访谈 |

#### 8.5 人格障碍

| 具体诊断 | 常见表现 | 推荐治疗取向 |
|---------|---------|-------------|
| **边缘型人格障碍** | 情绪不稳、人际混乱、自伤/自杀 | 辩证行为疗法（DBT）、心智化治疗（MBT）、图式治疗 |
| **回避型人格障碍** | 社交回避、自卑、孤立 | 图式治疗、CBT、团体治疗 |
| **依赖型人格障碍** | 过度依赖、无法独立、害怕分离 | 精神分析、图式治疗、CBT |
| **自恋型人格障碍** | 夸大自我、缺乏共情、脆弱自尊 | 精神分析、图式治疗、心智化治疗 |

#### 8.6 其他

| 具体诊断 | 常见症状 | 推荐治疗取向 |
|---------|---------|-------------|
| **躯体症状障碍** | 过度关注身体、反复就医、功能受损 | CBT、正念疗法、ACT |
| **睡眠障碍** | 失眠、嗜睡、睡眠质量差 | 失眠认知行为疗法（CBT-I）、睡眠卫生教育 |
| **物质成瘾** | 酒精/药物依赖、戒断症状 | 动机式访谈、12步疗法、CBT |

### 9. 自杀风险

| 具体情况 | 风险等级 | 推荐干预 |
|---------|---------|---------|
| **自杀意念** | 有想法但无计划 | 危机评估、安全计划、CBT |
| **自杀计划** | 有具体计划 | 危机干预、住院评估、密集跟进 |
| **自杀未遂** | 曾尝试自杀 | 住院治疗、DBT、家庭治疗 |
| **自残行为** | 非自杀性自伤 | DBT、情绪调节训练、CBT |

## 六、实施步骤

### 阶段一：建立目录结构（本周完成）

1. 创建 9 个一级分类文件夹
2. 创建二级子分类文件夹（如家庭关系下的 8 个子类）
3. 创建知识库目录结构

### 阶段二：填充知识库（1-2个月）

1. 选择 3-5 个最常用的治疗取向开始（如 CBT、精神分析、存在主义）
2. 为每个治疗取向编写：
   - 理论基础.md
   - 核心技术.md
   - 适用问题清单.md
3. 逐步补充其他治疗取向

### 阶段三：积累案例（持续进行）

1. 每次接访后，将记录归档到对应问题类型文件夹
2. 在案例中标注使用的治疗取向和技术
3. 定期回顾案例，提取最佳实践

### 阶段四：建立索引（3个月后）

1. 创建"问题类型 → 治疗方案"交叉索引
2. 例如：`抑郁症治疗方案汇总.md` 列出所有适用于抑郁症的治疗取向和对应案例
3. 建立快速检索系统

## 七、注意事项

### 1. 灵活性

- 一个案主可能涉及多个问题领域（如：抑郁症 + 家暴），选择**主要问题**归档
- 可以在档案元数据中标注**次要问题标签**

### 2. 保密性

- 所有案例严格脱敏
- 知识库不上传云端
- 定期备份到加密硬盘

### 3. 更新迭代

- 治疗取向和技术不断发展，知识库需定期更新
- 建议每季度回顾一次分类体系是否需要调整
