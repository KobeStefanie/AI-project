---

## 一、核心设计原则

### 1. 案主为中心
- 物理存储按**案主代号**组织
- 一个案主一个文件夹，包含全部治疗历史
- 支持 4-50 次的长期咨询追踪

### 2. 阶段分组
- 同一案主下，按问题转变划分阶段
- 每个阶段一个子文件夹
- 阶段内按时间顺序排列接访记录

### 3. 序号连续
- 接访记录序号全局递增（跨阶段连续）
- 方便统计总次数

### 4. 双维度索引
- 物理存储按案主组织
- 通过索引文件支持按"关系问题"和"精神症状"检索
- 单一存储，多维访问

## 六、工作流程

### 1. 新案主首次接访
1. 在"案主档案"下创建文件夹：`张XX`
2. 创建 `案主信息.md`，填写基本信息
3. 创建阶段文件夹：`阶段1-婆媳关系+焦虑症（2026.06-至今）`
4. 用接访记录模板创建：`01-20260614.html`
5. 接访后填写，保存

### 2. 后续接访
1. 打开案主文件夹，找到当前阶段文件夹
2. 打开上次记录（如 `03-20260628.html`），重读
3. 创建本次记录：`04-20260705.html`
4. 接访后填写，保存

### 3. 问题转变，开启新阶段
**判断标准**：连续3次主要谈新主题，或案主明确表达转向新目标

**操作步骤**：
1. 在案主文件夹下创建新阶段文件夹：  
   `阶段2-自我成长（2026.09-至今）`
2. 更新阶段1文件夹名称，加结束日期：  
   `阶段1-婆媳关系+焦虑症（2026.06-2026.08）`
3. 在 `案主信息.md` 中更新阶段概览，为阶段1写总结
4. 创建新记录：`16-20260905.html`（序号延续）

### 4. 定期维护索引（每月1号）
1. 回顾本月新增案主
2. 扫描所有 `案主信息.md` 的标签
3. 更新"关系维度索引"和"症状维度索引"
4. 备份全部档案到外部硬盘

## 八、阶段总结模板

每个阶段结束时，在阶段文件夹内创建 `阶段总结.md`：

```markdown
# 阶段1总结：婆媳关系+焦虑症

> 2026.06.14 - 2026.08.30 | 共15次咨询

---

## 治疗目标
1. 缓解焦虑症状（失眠、心悸、回避）
2. 改善婆媳关系，建立界限
3. 提升案主的情绪调节能力

## 主要治疗取向
- 家庭治疗（系统观）
- CBT认知行为治疗

## 关键技术应用
1. **认知重构**：识别灾难化思维、非黑即白
2. **暴露疗法**：逐步暴露婆媳互动场景
3. **家庭界限设定**：练习说"不"
4. **呼吸训练**：缓解急性焦虑

## 治疗轨迹

### 前期（1-5次）
- 建立关系，充分倾听
- 评估危机等级：中度
- 案主主要宣泄情绪，归罪婆婆

### 中期（6-10次）
- 认知重构开始生效
- 案主开始觉察自己的完美主义
- 尝试设定界限，但常失败

### 后期（11-15次）
- 界限设定成功次数增加
- 焦虑症状明显缓解
- 危机等级降至轻度
- 案主提出新需求："我发现我总是在满足别人，我是谁？"

## 结果评估
- ✅ 治疗目标1：达成（焦虑症状减轻70%）
- ✅ 治疗目标2：部分达成（关系改善，但仍有冲突）
- ✅ 治疗目标3：达成（掌握基本情绪调节技术）

## 未完成议题
- 深层自我认同问题（转入阶段2）
- 与丈夫的关系动力（未充分探索）

## 督导建议
- （如有督导，记录建议）

## 咨询师自我反省
- 前期过度同情案主，可能强化了"受害者"认同
- 中期转向认知重构后，案主进展明显
- 需要更多关注夫妻子系统

## 后续计划
- 转入阶段2：自我成长
- 治疗取向调整为：存在主义 + 人本主义
```
