# 心理咨询案例管理系统（4-心理咨询-S1）

> 创建于 2026-05-31 | 当前版本 v1.0 | 最后更新 2026-06-23

## 📋 项目定位

**本地化、AI驱动的心理咨询案例管理系统**

将接访记录转化为结构化案例库，支持多维度检索和AI督导分析。纯个人使用，随能力成长而进化。

**当前体系**：大观学派（希望热线危机干预） | **后续扩展**：CBT → 精神动力学 → 人本主义 → 存在主义

---

## 🏗️ 系统架构

### 三层架构

```
Layer 1: 原始文档（Word）
   ↓  AI处理 + 人工审核
Layer 2: 结构化数据（JSON）+ 四大索引
   ↓  生成HTML
Layer 3: 案例库可视化（HTML）
```

**数据流**：
1. Word接访记录 → `data/cases/original/`
2. Python脚本处理 → AI分析 → 生成JSON → `data/cases/processed/`
3. 更新4个索引 → `data/index/`
4. 生成HTML案例库 → `D:\心理咨询师\2-知识库\治疗取向\大观学派\`

## 🏷️ 核心概念

### 案例编号规则
**格式**：`C + YYYYMMDD + 3位序号`
- 日期来源：文件名中的日期（非接访日期）
- 示例：`C20260616001`（2026年06月16日第1个案例）

### 危机等级（大观学派）
| 等级 | 名称 | 定义 |
|------|------|------|
| **S** | 自杀风险 | 有明确计划/手段/遗书/失联 |
| **L** | 生命危险 | 他杀风险/严重自伤/精神病性症状 |
| **M** | 中度危机 | 抑郁/焦虑影响功能/创伤急性期 |
| **C** | 慢性困扰 | 长期关系问题/自我成长议题 |
| **Z** | 正常范围 | 偶发情绪波动/一般性困惑 |

### 四大索引
1. **关系标签**：家庭关系、职场关系、社交关系、自我关系
2. **症状标签**：情绪障碍、焦虑障碍、创伤相关、行为问题、人格障碍
3. **危机等级**：S/L/M/C/Z五级分类
4. **关键词**：从案例中提取的核心关键词

## 📌 设计原则

- **本地优先**：数据不出本机，保护隐私
- **AI辅助**：AI提供分析建议，最终由咨询师确认
- **原始保留**：Word文档永久保存，JSON独立存储
- **可进化**：随用户能力成长而扩展（大观→CBT→精神动力学→...）
- **纯个人**：为个人学习和成长设计，不考虑多人协作
