# 统一标签库使用说明

> 创建时间：2026-06-26  
> 版本：v1.0

---

## 📋 问题背景

**原问题**：接访记录模板和案例库的标签各自独立，不统一。

- **接访记录模板**：硬编码的标签选项
- **案例处理脚本**：AI 动态提取标签
- **案例库展示**：从索引读取标签

三者标签不一致，导致数据混乱。

---

## 🎯 解决方案：统一标签库

创建 `data/config/tags_library.json` 作为**唯一标签权威源**，所有模块从此读取。

---

## 📁 标签库结构

### 1. 关系标签（relation_tags）

格式：`{一级分类}-{二级分类}-{具体标签}`

**示例**：
```
家庭关系-亲子关系-父母缺位
社交关系-友谊-青少年同伴支持
工作关系-企业经营-企业经营压力
```

**5大一级分类**：
- 家庭关系（👨‍👩‍👧‍👦）
- 社交关系（👥）
- 工作关系（💼）
- 自我关系（🧘）
- 社会关系（🌍）

### 2. 症状标签（symptom_tags）

格式：`{一级分类}-{二级分类}-{具体标签}`

**示例**：
```
情绪障碍-抑郁症-重度（已确诊）
行为问题-自杀风险-自杀计划
精神症状-幻听
```

**7大一级分类**：
- 情绪障碍（😔）
- 情绪问题（😰）
- 精神症状（🌀）
- 行为问题（⚠️）
- 躯体症状（🩺）
- 内分泌（🧬）
- 创伤相关（💔）

### 3. 危机等级（crisis_levels）

**5级分类**：
- **S** - 自杀风险（红色）
- **L** - 生命危险（橙色）
- **M** - 中度危机（黄色）
- **C** - 慢性困扰（蓝色）
- **Z** - 正常范围（绿色）

### 4. 技术库（techniques）

**4大学派**：
- 大观学派（11项技术）
- CBT（8项技术）
- 精神动力学（5项技术）
- 人本主义（4项技术）

### 5. 人群分类（age_groups）

**5个年龄段**：
- 儿童（0-12岁）
- 青少年（13-18岁）
- 青年（19-35岁）
- 中年（36-55岁）
- 老年（56岁以上）

## 🔧 使用方式

### 1. 接访记录模板
**未来改造**：从 `tags_library.json` 动态加载标签选项，替代硬编码。

### 2. 案例处理脚本
**AI 验证**：AI 提取的标签需与标签库匹配，不在库中的标记为"需审核"。

### 3. 案例库展示
**读取标签库**：展示时使用标签库的图标、颜色、分类信息。

### 4. 标签同步工具
**`src/sync_tags.py`**：检查现有索引标签是否都在标签库中，发现新标签时提示添加。

## 🛠️ 维护规则

### 1. 添加新标签
在 `data/config/tags_library.json` 中添加，格式：`{一级}-{二级}-{具体}`

### 2. 修改标签
修改标签库后，运行 `python src/sync_tags.py` 检查一致性。

### 3. 标签命名规范
- **层级分隔符**：使用 `-` 连接
- **括号说明**：可用括号补充说明，如 `抑郁症-重度（已确诊）`
- **避免重复**：不同层级避免同名，如不要出现 `行为问题-自杀风险` 和 `行为问题-自杀风险-自杀风险`

### 4. 定期同步
处理新案例后，运行同步工具检查是否有新标签需添加。
