### 🔄 Phase 2: 改造案例处理脚本（下一步）

**目标**：AI 提取的标签需与标签库验证匹配

**修改文件**：`src/case_processor.py`

**改造内容**：
1. 加载标签库
2. AI 提取标签后，验证是否在标签库中
3. 不在库中的标签标记为 `[需审核]`
4. 生成处理报告时提示哪些标签需要添加到标签库

**伪代码**：
```python
def validate_tags(ai_extracted_tags, tags_library):
    valid_tags = []
    invalid_tags = []
    
    for tag in ai_extracted_tags:
        if tag in tags_library:
            valid_tags.append(tag)
        else:
            invalid_tags.append(tag)
    
    return valid_tags, invalid_tags
```
