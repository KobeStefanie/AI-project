# 压缩对话实录

本文件夹存储Context压缩前的对话整理记录，用于压缩后恢复上下文。

## 文件命名规范

格式：`YYYY-MM-DD_HH-MM_主题描述.md`

示例：
- `2026-06-29_16-30_流派配置修复.md`
- `2026-06-29_18-00_数据结构迁移.md`

## 文件内容结构

每个实录文件应包含：

1. **关键上下文**：当前状态、重要决策、关键文件
2. **详细记录**：具体工作内容和代码变更
3. **下一步计划**：压缩后需要继续的工作

## 使用方式

### 压缩前
- Token使用接近上限时（如180k/200k）
- 主动创建实录文件，整理当前对话

### 压缩后
- 立即读取最新的实录文件
- 恢复上下文，继续之前的工作

详细规则参见：`D:/AI-项目/0-config/memory/context-compact-workflow.md`
