# 删除案例说明

## 使用交互式删除工具（推荐）

```bash
cd "D:\AI-项目\4-心理咨询-S1"
python src/delete_visitor_interactive.py
```

这个工具会：
1. 列出所有来访者及其基本信息
2. 让你选择要删除哪些（输入序号，多个用空格分隔）
3. 二次确认后执行删除
4. 自动删除数据文件和生成的HTML页面

## 删除单个案例（命令行）

```bash
cd "D:\AI-项目\4-心理咨询-S1"
python src/api_delete_visitor.py V20260629001
```

## 删除后重新生成页面

```bash
python src/generate_visitor_library.py
```

## 手动删除

如果需要手动删除，需要删除以下两个位置：

1. **数据文件**：`data/visitors/{visitor_id}/`
2. **生成的HTML**：`output/来访者库/{visitor_id}/`

删除后记得重新生成来访者库页面。
