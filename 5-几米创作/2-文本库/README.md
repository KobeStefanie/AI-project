> 项目共享资料。所有作品的文案统一归档于此，永不删除。

## 目录结构

```
1-文案库/
├── README.md           # 本文件 — 文案库索引
├── <作品名>/           # 按作品分目录
│   ├── 旁白/           # 叙事旁白
│   ├── 对白/           # 角色对话
│   ├── 描述/           # 画面/场景描述
│   ├── 脚本/           # 分镜脚本
│   └── 标题与宣传/     # 标题、宣传语、简介
└── 灵感碎片/           # 未归属到具体作品的零散文案
```

## 使用规则

1. 所有文案**永不删除**，被替换的旧版本标注"已替换"并保留
2. 每段文案文件头部包含元信息（作品、类型、场景、状态、日期、关联版本）
3. 版本命名：`<场景/用途>-<类型>-v<版本>.md`，以 Markdown 格式存储
4. 灵感碎片定期整理，能归属到具体作品的及时归入对应目录
5. 本 README 在每次添加/修改文案后同步更新索引
