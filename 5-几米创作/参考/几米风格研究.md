## 一、三大时期风格速览

| | 早期（1998-2000） | 中期（2001-2004） | 后期（2005-至今） |
|------|------|------|------|
| 色彩 | 黑白灰无彩色 | 灰蓝冷调 | 高彩度缤纷浓烈 |
| 情绪 | 绝望、虚无、孤独 | 疏离、批判、错失 | 温暖、疗愈、希望 |
| 代表作 | 向左走向右走、月亮忘记了 | 地下铁、又寂寞又美好 | 星空、时光电影院 |

## 二、核心视觉 DNA

- **小人物大背景**：主角占画面 5-10%，被辽阔空间包围
- **冷暖过渡**：灰蓝冷调 → 暖黄微光，几米的标志性情绪曲线
- **超现实元素**：隐藏的巨型动物、漂浮的月亮、游在空中的鱼
- **东方线条**：简约优雅，水彩湿画法晕染
- **大面积留白**：传递纯洁、空寂的境界感

## 三、核心叙事 DNA

- **以图为主、字为辅**：文字精简如格言
- **第一人称自述**：增强代入感
- **孤独→希望**：从不回避孤独，但总留一扇窗
- **碎片化诗意**：留白跳跃，不解释太多

## 四、符号系统

| 元素 | 象征 |
|------|------|
| 月亮 | 孤独、思念、夜晚的慰藉 |
| 鱼/鱼缸 | 未说出口的情绪、自由与束缚 |
| 地铁/火车 | 人生旅程、离别与追寻 |
| 雨伞/雨天 | 都市孤独、低谷期 |
| 窗台/窗户 | 内心与外界的观望 |
| 猫 | 独立、神秘、都市温情 |
| 气球 | 飘走的缘分、脆弱的希望 |

## 五、创作公式

> 几米风 = 小人物 + 大城市 + 冷蓝色调开场 + 超现实陪伴物 + 一句话诗意字幕 + 暖黄微光收尾
