## 一、核心定位：一个让人透口气的空间

### 探索过程

```
v1 阶段：
  心理咨询师 → 导流账号 → 唤醒内在小孩 → 几米风格 → 引发共情

v2 突破：
  "治愈"暗含"你有病需要被治好"
  → 不要治愈，只要一个空间
  → 没有评判，只有被看见
  → 没有规劝（不向善、不归恶），只是一个可以透口气的地方

v3 深化：
  "被看见"仍然暗含一个主动去"看"的人
  → 不是"你去看见他"，是他自己看见自己
  → 画面是镜子，文字是镜框
  → 他不觉得有人在理解他——他自己理解了
```

### 最终定位

> **一个用几米视觉语言构建的心理安全空间。**
>
> 这里没有答案。我给不了答案——很多事情本来就没有答案。
> 我能做的就是给一个空间，让对方在这里恢复能量，或者看见自己。
> 喘过来了，自己继续向前。
> 哪怕不是治愈。

### "呼吸感"的五个创作原则

1. **不"给"** — 不给安慰、不给建议、不给解决方案。只提供一个不催促的空间。
2. **不评判** — 不对观众的情绪做任何价值判断。不暗示"你应该好起来"。
3. **大量留白** — 画面 80% 是安静的蓝灰色，文字极简或空白。留白不是空，是**呼吸的必要条件**。
4. **不催促的节奏** — 视频留几秒静止画面，那是观众呼气的时间。快节奏 = 需要跟上 = 压力。
5. **中性的情绪温度** — 不冷到确认绝望，不暖到推销幸福。5-15°C 刚好。

### 内容发布前唯一需要过的问题

> "这条内容，是在给观众一个透口气的空间，还是在给他一个需要回应的东西？"

如果是后者，就偏了。

## 二、情感逻辑 v3

### v1 → v2 → v3 的进化

```
v1: 看见伤口 → 承认它的存在 → 陪伴 → 微小的可能性
v2: 看见 → 陪一会儿 → 呼。
v3: 路过 → 停下来 → 自己发现 → 呼。
```

### v3 的关键突破

v2 的「看见」暗示有一个**主动去看的人**（账号/咨询师）。观众刷到内容时心理防御是满的——标题党、贩卖焦虑、教你做人，都是他们正在躲避的东西。一上来就试图"看见"他，他会本能后退：「你又想对我做什么？」

**在「看见」之前，需要一个更轻的步骤：什么都不做。**

```
路过 → 停下来（被画面吸引，没有目的）→ 自己发现（不是被告知）→ 呼。
```

关键差异：**不是你去看见他，是他自己看见自己。**

画面只是一面镜子。文字只是镜子的边框。他不觉得有人在"理解"他——他自己理解了。

这比「被看见」更安全，因为没有任何人闯入他的空间。

### 与心理咨询的对齐

咨询师不会告诉来访者"你应该好起来"。
咨询师做的事情是：**创造一个足够安全的空间，让来访者可以自由地呼吸、自由地成为自己。**

这个账号做的是同一件事——用画面、文字、声音构建那个空间。

### 每个人内心都是向阳的

> 每一个人内心都是向阳的。只是他的内心动能被太多压抑、敏感所损耗了。
> 我们要做的就是给他一个空间，让他自己恢复。
> 可能等一等，可能缓一缓。

## 三、账号身份

### 名字：角落先生

经过 7+ 候选的深度讨论和情绪温度对比，最终确定：

| 候选 | 类型 | 情绪温度 | 结果 |
|------|------|---------|------|
| 未完成 | 哲学概念 | 8°C — 偏冷、哲学感 | ❌ |
| 未完成的画 | 意象 | 10°C — 诗意、与绘画形式统一 | ❌ |
| 未完成先生 | 人格化 | — 贬义歧义 | ❌ |
| 角落的猫 | 生命化空间 | 15°C — 最高传播力，与小孩主体冲突 | ❌ |
| 留白 | 视觉哲学 | 10°C — 文人感，设计感 | ❌ |
| 窗边 | 空间/位置 | 12°C — 平淡 | ❌ |
| 不着急 | 口语/态度 | 14°C — 温暖，像口头禅 | ❌ |
| 深海里的鲸 | 深处的存在 | 3°C — 太冷太沉 | ❌ |
| **角落先生** | **人格化空间** | **15°C** — 有人味、温暖、守护、和小孩自然陪伴 | **✅** |

> **"角落先生"的核心优势：** 先生和小孩之间有自然的守护关系——不是父亲、不是老师、不定义身份，只是角落里的那个人。与几米「躲进世界的角落」DNA 一致。

### 简介

> **一个角落，门开着。在听。**

| 元素 | 作用 |
|------|------|
| 一个角落 | 呼应名字，确定空间 |
| 门开着 | 安全感，不锁门 |
| 在听 | 不是「倾听者」（头衔/身份），是「在听」（正在发生的动作）。咨询师的核心工作。不推销、不宣告、只是在 |

### 认证标签

- 简介不出现「心理咨询师」——太硬，和「不推销」矛盾
- 认证标签和简介同一屏，观众自己会连接
-「在听」→ 后来发现是心理咨询师 →「难怪你这么懂」而非「果然是专业的」

## 四、文字风格指南

> 用户原文：**简短、自然、对话感、画面感。**

| 原则 | 说明 | 反例 |
|------|------|------|
| **简短** | 不是少，是刚好。多了就是在抢观众呼吸的空间 | 长篇大论、鸡汤段落 |
| **自然** | 不说书面语，不写"金句"。像一个人坐在窗边随口说的话 |"愿你被世界温柔以待" |
| **有对话感** | 像在和观众说话，不是一个人在讲。观众读起来像听到了声音 | 解说腔、教育腔 |
| **有画面感** | 文字本身是一幅小画。和画面之间有缝隙，观众在缝隙里看到自己 | 纯抽象描述、说教 |

**已有实践（「凌晨三点的人」脚本）：**
- 「整座城市都睡了」
- 「你和你的恐惧还醒着」

→ 简短、自然、像有人在你旁边轻声说的、看到字的时候脑子里已经有画面。

## 五、IP 策略：风格 IP，不是角色 IP

**这个账号的 IP 是一种风格，不是一个具体的人物。**

| | 角色 IP（如寂地的V先生） | 风格 IP（角落先生） |
|------|--------------------------|---------------------|
| 识别方式 | "又是这个小男孩" | "这是那个几米风的号——可以透口气的地方" |
| 承载能力 | 一个角色承载不了所有情绪状态 | 风格可以适配任何情境、任何内在小孩 |
| 品牌积累 | 观众记住这张脸 | 观众记住**这种感觉**——刷到就知道是你 |

### 风格 IP 的五层构成

1. **色彩识别** — 灰蓝 + 暖黄的固定调色板，看到画面不用看账号名就知道是谁
2. **构图识别** — 大场景小人物、背影/侧影、大量留白
3. **节奏识别** — 视频的呼吸感，不催促的停顿
4. **文字识别** — 不评价不引导的极简文字（或完全无文字）
5. **情绪温度识别** — 5-15°C，不冷到绝望也不暖到推销

## 六、账号视觉系统

### 头像：墙角+猫窝+蓝灰猫（当前使用）

> 暖灰米色墙角 → 地上一个圆形猫窝，暖米色软边 → 一只蓝灰色肥猫蜷在窝里，身体盘成松软的圆弧 → 猫和猫窝是画面最暖的一小片 → 墙角线几笔淡墨，其余水彩大面积柔和晕染

- 文件：`角落先生-头像-v8.png`（1024×1024）
- 符号逻辑：圆形猫窝 + 蜷缩的猫 = 被包裹的安全感。缩到 60px 仍能看到暖灰中一点深灰蓝——那是猫，是角落里安静存在的生命
- 设计迭代：v7（C2 极简光+微小剪影）→ 圆形裁剪导致人物不可见 → 窗台+猫（不像几米笔触）→ 墙角+猫窝+远镜头（太冷）→ 中等景别暖调最终版
- 几米笔触关键词：松散墨线勾勒、彩铅细密排线、水彩湿画法晕染、纸纹可见

### 封面：暖灰暮光（当前使用）

> 暖灰米色墙面 → 木门半开 → 暖金色光从门缝漏出 → 猫蹲在台阶上 → 小盆绿植 → 大量留白

- 文件：`角落先生-封面-v9.png`（1024×1024）
- 与头像统一暖灰暮光色调
- 门+光+猫=「角落先生」的完整视觉符号

## 九、导流策略：空间本身就是吸引力

### "内容公司"定位

> **我们本质上是一个内容公司。好的内容可以起到自然的导流效果。**

- 不是"做账号引流"，是"做好内容，自然有人来"
- 不主动设计转化路径、不在内容里做 call to action
- "透口气"本身就会让人想说话——观众会在评论区说出内心的话语和情绪
- 当他们说出来，连接就已经发生了

### 自然导流链

```
看到内容 → 松了口气 → 「这是谁做的？」
    → 点进主页
    → 看到「角落先生」+「一个角落，门开着。在听。」
    → 看到认证「心理咨询师」→ 自己连接起来
    → 关注 / 私信 / 自己决定要不要进一步
```

### 为什么简介不写「心理咨询师」

- 写在简介里 = 在门口挂牌子 = 触发防御
- 认证标签和简介同一屏 = 观众自己发现 = 信任更深
- 「在听」已经为认证埋了桥——发现是咨询师时不会突兀

## 十一、6 步创作流程

> **所有子项目严格按照 6 步推进，每步确认后进入下一步。**

| 步骤 | 名称 | 产出物 |
|------|------|--------|
| 1 | 写故事或剧本 | 故事大纲 / 剧本 |
| 2 | 分镜及脚本 | 分镜脚本（含画面描述、旁白/对白） |

| 4 | 根据图片生成视频 | 视频片段 / 成片 |
| 5 | 根据视频生成音效和配音 | 音频素材 |
| 6 | 后期剪辑 | 最终成片 |
