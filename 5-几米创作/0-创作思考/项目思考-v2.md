## 一、核心定位：一个让人透口气的空间

### 探索过程

```
v1 阶段：
  心理咨询师 → 导流账号 → 唤醒内在小孩 → 几米风格 → 引发共情

v2 突破：
  "治愈"暗含"你有病需要被治好"
  → 不要治愈，只要一个空间
  → 没有评判，只有被看见
  → 没有规劝（不向善、不归恶），只是一个可以透口气的地方
```

### 最终定位

> **一个用几米视觉语言构建的心理安全空间。**
>
> 这里没有答案。我给不了答案——很多事情本来就没有答案。
> 我能做的就是给一个空间，让对方在这里恢复能量，或者看见自己。
> 喘过来了，自己继续向前。
> 哪怕不是治愈。

### 与普通治愈系内容的本质区别

| 普通治愈系 | 这个账号 |
|-----------|---------|
| "你会好起来的" | "你现在这样也是可以的" |
| 给答案 | 不假装有答案 |
| 帮你解决问题 | 陪你坐一会儿 |
| 目标是"好了" | 目标是"喘过来了" |
| 内容在发光 | 内容在留白 |
| 观众被感动 | 观众松了口气 |

### "呼吸感"的五个创作原则

1. **不"给"** — 不给安慰、不给建议、不给解决方案。只提供一个不催促的空间。
2. **不评判** — 不对观众的情绪做任何价值判断。不暗示"你应该好起来"。
3. **大量留白** — 画面 80% 是安静的蓝灰色，文字极简或空白。留白不是空，是**呼吸的必要条件**。
4. **不催促的节奏** — 视频留几秒静止画面，那是观众呼气的时间。快节奏 = 需要跟上 = 压力。
5. **中性的情绪温度** — 不冷到确认绝望，不暖到推销幸福。5-15°C 刚好。

### 内容发布前唯一需要过的问题

> "这条内容，是在给观众一个透口气的空间，还是在给他一个需要回应的东西？"

如果是后者，就偏了。

## 二、IP 策略：风格 IP，不是角色 IP

### 关键认知

**这个账号的 IP 是一种风格，不是一个具体的人物。**

| | 角色 IP（如寂地的V先生） | 风格 IP（这条路） |
|------|--------------------------|---------------------|
| 识别方式 | "又是这个小男孩" | "这是那个几米风的号——可以透口气的地方" |
| 承载能力 | 一个角色承载不了所有情绪状态 | 风格可以适配任何情境、任何内在小孩 |
| 品牌积累 | 观众记住这张脸 | 观众记住**这种感觉**——刷到就知道是你 |
| 典型参照 | 吾皇万睡、V先生 | **几米本人**——画过无数角色，翻开就知道是几米 |

### 风格 IP 的五层构成

1. **色彩识别** — 灰蓝 + 暖黄的固定调色板，看到画面不用看账号名就知道是谁
2. **构图识别** — 大场景小人物、背影/侧影、大量留白、窗框门框的二次框架
3. **节奏识别** — 视频的呼吸感，不催促的停顿，留几秒不动的静止画面
4. **文字识别** — 不评价不引导的极简文字（或完全无文字）
5. **情绪温度识别** — 5-15°C，不冷到绝望也不暖到推销

### 内在小孩的视觉化策略

- **不是固定的一个小孩**，而是情境驱动的不同小孩
- 每个子项目的主题决定那个小孩的样子：
  - 被压抑的、被误解的、被质疑的
  - 渴望被看见的、渴望证明自己的
  - 多愁善感的、沉默不语的
- 观众在不同内容里认出**不同的自己**

## 三、导流策略：空间本身就是吸引力

### 核心原则：不设计转化路径

- 不主动引导关注、不推销咨询、不在内容里做任何 call to action
- "透口气"本身就会让人想说话——观众会在评论区说出内心的话语和情绪
- 当他们说出来，连接就已经发生了
- 他们会自己点进主页，看到"心理咨询师"——自己连接起来

### 自然导流链

```
看到内容 → 松了口气 → 觉得"这个人懂我"
    → 在评论区说出自己的情绪
    → 点进主页
    → 看到"心理咨询师"→ 自己决定要不要进一步
```

### 主页简介方向

- 不用"心理咨询，请联系XXX"
- 更轻的表达："陪你一起，看见你的内在小孩"
- 或更直接地呼应定位："一个可以透口气的地方"

## 四、对标账号与竞争格局

### 直接竞品：接近为零

> 目前在抖音和小红书上，**没有人把"几米式水彩绘本风格"和"心理学深度"结合在一起做短视频。**

### 最接近的三类账号

| 类型 | 代表账号 | 做了什么 | 缺了什么 |
|------|----------|---------|----------|
| 治愈系动画 | @柏物语（100万粉）、@鱼书书书 | AI 动画、梦核拼贴、可爱角色 | 没有心理学深度 |
| 心理学科普 | 小红书大量心理咨询师号 | 口播+文字卡片讲"内在小孩" | 是"讲课"不是"共情" |

### 三个值得研究的账号

|------|-----------|---------------|

## 五、五位画家风格深度拆解

### 5.1 几米 (Jimmy Liao) — 日常基调，80% 内容的底色

**地位**：主参考，画面 DNA 的起点

#### 色彩体系

| 参数 | 特征 | AI 提示词转化 |
|------|------|-------------|
| 主色调 | 青蓝、灰蓝为主，整体灰调 | `deep blue-gray tones, muted palette` |
| 高光色 | 暖黄、淡金——极克制地使用 | `a thin sliver of warm amber light` |
| 饱和度 | 低彩度，颜色"含灰"，不艳丽 | `desaturated, dusty colors` |
| 对比方式 | 冷暖对比（大面积冷蓝+小面积暖黄） | `cold blue dominant, single warm accent` |
| 底色 | 灰色调打底，颜色叠加在灰上 | `gray undertone throughout` |
| 红色用法 | 极少用纯红，用时极克制（如一把红伞） | 红色面积不超过画面 3% |

#### 构图铁律

| 参数 | 特征 | AI 提示词转化 |
|------|------|-------------|
| 人景比例 | 人物占画面 5%-15% | `tiny figure in vast space, less than 10% of frame` |
| 视角 | 平视或大俯角，极少仰视 | `eye-level or high-angle bird's-eye view` |
| 人物朝向 | 背对/侧对镜头为主 | `figure seen from behind, facing away` |
| 空间法则 | "疏可跑马，密不透风" | `large empty negative space` |
| 框中之框 | 窗框、门框、镜子二次压缩人物 | `figure framed by a window frame` |

#### 人物造型

- 眼睛是两个小点，或完全无五官
- 头身比偏 Q（非写实）但不幼稚
- 线条：短而碎的竖线排列构成面（如头发、毛衣纹理）
- 轮廓：用 1.0 油性笔勾勒，再以水彩染色

#### 技法

- **媒材**：水彩 + 水溶性彩铅 + 针笔，坚持手绘
- **背景**：湿画法（先打湿纸再罩色→自然晕染）
- **前景**：干湿结合
- **偶然性**：水色流动的自然效果，不被完全控制

### 5.2 陈志勇 Shaun Tan — 极端压抑时借用

**地位**：《The Red Tree》是"不治愈的治愈风"教科书

#### 色彩体系

| 参数 | 特征 | AI 提示词转化 |
|------|------|-------------|
| 主色调 | 灰暗、压抑的工业色调——混凝土灰、铁锈褐 | `industrial gray-brown, oppressive muted earth tones` |
| 高光色 | **红色**作为唯一的"希望密码" | `a single red leaf as the only warm accent` |
| 饱和度 | 比几米更暗更重，接近油画质感 | `oil-painting texture, heavy impasto feel` |
| 对比方式 | **红 vs. 黑/灰**二元对立 | `monochrome grays with ONE red element` |

#### 构图与空间

- 比例失调：巨大物体压迫渺小人物
- 封闭感：玻璃瓶、被锁的窗
- 拼接感：拼贴技法、破碎文字碎片

#### 超现实符号系统

| 情绪 | 视觉化 |
|------|--------|
| 压抑/忧郁 | 巨大深海怪鱼漂浮在城市上空 |
| 无人理解 | 扩音器中掉落的零散字母 |
| 被困住 | 小女孩在玻璃瓶内，戴氧气罩 |
| 希望始终存在 | 每一页隐藏的小红叶（需要找才发现） |

### 5.3 寂地 (Jidi) — 符号化角色策略 + 光晕技法

**地位**：中国本土最接近的参照，"情感色彩魔术师"

#### 色彩演变

| 阶段 | 色调 | 心境 |
|------|------|------|
| 早期（1-2册） | 灰暗凝重 | 成都灰调+母亲去世 |
| 中期（3-4册） | 逐渐温暖清新 | 《春暖花开》转折 |
| 后期（7-8册） | 明快温暖 | 《星辉谷》成熟 |

#### 核心视觉特征

| 参数 | 特征 | 对你的启发 |
|------|------|-----------|
| 色彩 | "在昏暗底调上加入光的晕染" | 几米"灰蓝+暖黄" + 寂地"昏暗+光晕"=可融合 |
| 人物 | 早期无五官，眼睛是"深邃黝黑的空洞" | 和你的无五官/背影策略一致 |
| 符号 | V先生——黑色礼帽、心口红心，贯穿全系列 | 证明了固定符号的力量——但不等于你要复制 |

### 5.4 Tove Jansson（托芙·扬松）— 积极的孤独哲学

**地位**：给出了"独处但不孤单"的正面版本

#### 孤独谱系——每个角色是一种孤独

| 角色 | 孤独类型 | 对应内在小孩状态 |
|------|---------|----------------|
| 史力奇 | 主动选择独处，"需要孤独" | "我选择一个人，这很好" |
| 哥谷 (Groke) | 寒冷、渴望温暖却把人冻住 | "我想靠近但我害怕" |
| 菲利钟克 | 努力填补他人缺席的空白 | "我不能停下来" |

#### 视觉特征

| 参数 | 特征 | 对你的启发 |
|------|------|-----------|
| 线条 | 极简流畅，北欧设计式利落 | 几米碎线密集 + 杨松简洁轮廓 = 可融合 |
| 色彩 | 淡雅柔和，四季分明的自然色调 | 色彩随情绪变化 |
| 自然 | 大海、岛屿、森林是治愈场域——**不是背景是角色** | "内在小孩"不只在房间里，也可以在自然中独处 |
| 哲学 | "门永远对寻求庇护的人敞开" | 和咨询师身份高度契合 |

### 5.5 Rébecca Dautremer（海贝卡·朵特梅）— 轻盈的诗意之光

**地位**：提供了"释放后的轻盈感"的视觉语言

#### 核心视觉特征

| 参数 | 特征 | 对你的启发 |
|------|------|-----------|
| **光影** | 年轻时梦想当摄影师——画面有**柔焦、景深、逆光** | 几米和陈志勇都没有的**电影级打光感** |
| **红色** | 大面积暗红是签名色——可以是危险也可以是生命力 | 红色不只是"希望"，也可以是**活着的感觉** |
| **漂浮感** | "只要我想，就可以让任何东西飞起来" | 超现实不是压制的（陈志勇），也可以是**轻盈诗意的** |
| **媒材** | 不透明水彩 (gouache)，纸上先涂油脂再小笔触 | 颜色浓郁但不厚重 |
| **构图** | 圆形/三角形框架、"小剧场"式构思 | 更丰富的形式语言 |

### 5.7 AI 生成实操：调色板矩阵

按内在小孩的情绪状态切换视觉配方：

```
内在小孩状态 → 对应视觉配方

躲藏/恐惧    → 几米基调 + 陈志勇封闭空间 + 寂地光晕
沉默/压抑    → 几米灰蓝 + 陈志勇红黑对立（一片红叶在灰色房间）
等待/渴望    → 几米空旷 + 朵特梅逆光（人物坐窗边，光从背后穿过）
愤怒/被侵犯  → 陈志勇压迫感 + 寂地哥特（大量暗色+人物极小）
好奇/探索    → 几米淡彩 + 杨松自然治愈（人物站在森林/海边边缘）
被看见/连接  → 朵特梅暖金逆光 + 杨松陪伴感（两个小人对视，光包围）
释放/轻盈    → 朵特梅漂浮感 + 杨松自由自然（头发飞起，衣服飘动）
```

### 5.8 AI 生成分层提示词

#### 基础层（每次都有的）

```
watercolor and colored pencil on textured paper,
hand-drawn feel, visible paper grain,
soft bleeding color edges, wet-on-wet wash technique,
tiny figure in vast space, figure seen from behind,
muted desaturated palette, gray undertone
```

#### 情绪层（按需要切换）

**几米模式（日常陪伴感）：**
```
deep blue-gray dominant, single warm amber accent,
large negative space, figure framed by window or doorway,
atmospheric, breathable, quiet melancholy
```

**陈志勇模式（极端压抑感）：**
```
oppressive industrial gray-brown, suffocating atmosphere,
one single red element as hope symbol,
claustrophobic space, figure trapped or enclosed,
oil-painting weight, textured impasto
```

**朵特梅模式（轻盈释放感）：**
```
warm golden backlight, soft focus effect,
floating hair and fabric, wind-blown movement,
gouache texture, rich but not heavy colors,
theatrical lighting, shallow depth of field
```

## 六、内容的情感逻辑 v2

### 旧逻辑（v1）

```
看见伤口 → 承认它的存在 → 陪伴 → 微小的可能性
```

### 新逻辑（v2）：更简洁，更符合"呼吸感"

```
看见 → 陪一会儿 → 呼。
```

不需要"微小的可能性"——那个"可能性"本身也是一种期待、一种催促。

观众不需要"好了一点点"。观众只需要——

> 在这里，什么都不用做，什么都不用改变。
> 喘口气。
> 然后你想走就走，想留就留。

### 与心理咨询的对齐

咨询师不会告诉来访者"你应该好起来"。
咨询师做的事情是：**创造一个足够安全的空间，让来访者可以自由地呼吸、自由地成为自己。**

这个账号做的是同一件事——用画面和文字构建那个空间。
