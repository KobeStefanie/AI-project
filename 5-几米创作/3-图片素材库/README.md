# 几米创作 — 画面素材库

> 项目共享资料。所有作品的 AI 生成图片素材统一归档于此，永不删除。

## 目录结构

```
2-画面素材库/
├── README.md           # 本文件 — 画面素材库索引
└── <作品名>/           # 按作品分目录
    ├── <作品名>-<场景>-v<N>.png
    ├── <作品名>-<场景>-v<N>.prompt.md
    ├── 生成记录.md
    └── ...
```

## 使用规则

1. 所有图片素材**永不删除**，无论好坏一律保留
2. 版本命名：`<作品名>-<场景>-v<N>.png`（如 `小孩的独白-a-v1.png`）
3. 每张图片必须有同名 `.prompt.md` 归档（提示词、模型参数、分镜上下文、生成历史）
4. 每次生成后在 `生成记录.md` 追加登记
5. 本 README 在每次添加素材后同步更新索引
