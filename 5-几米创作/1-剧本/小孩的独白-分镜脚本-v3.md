## 声音设计

| 参数 | 设计 |
|------|------|
| 配乐 | 极简钢琴独奏，单音为主，八音盒质感 |
| 音量 | 低于独白，留白处（无旁白时）微微浮上来 |
| 环境音 | 无 |
| 关键音点 | 分镜 5「永远都不会错」— 钢琴一个低音；分镜 8 猫蹭腿时 — 钢琴停半拍，然后一个暖音；尾帧 — 静音 |

**配乐情绪曲线**：

```
分镜 1-2：八音盒式单音，3-4 个音循环，轻
分镜 3-4：钢琴进入，缓慢下行音阶，留白处浮上来
分镜 5-6：钢琴最弱，几乎只有独白
分镜 7-8：钢琴回来，比之前更暖，分镜 8 猫蹭过来时一个暖音
尾帧：静音
```

## 剪辑指导

### 画面切换

| 过渡 | 方式 | 时长 |
|------|------|------|
| 分镜 1→2→3→4→5→6→7→8 | 淡出淡入 | ~0.5s 每次 |
| 分镜 8 → 尾帧 | 淡出到黑底 | ~1s |

### 画面节奏

- 所有分镜共享米白色背景，切换时背景不变，只有人物位置/姿态变化——视觉上是「同一个空间里的 8 个瞬间」
- 每个画面做极慢的 Ken Burns（幅度极小，几乎察觉不到），不要静止死图
- 场景切换处的 0.5s 淡入淡出，观众几乎感觉不到切换

### 字幕样式

| 参数 | 要求 |
|------|------|
| 字体 | 思源黑体 Light 或类似细体无衬线 |
| 颜色 | 深灰色（#333333），不用纯白——米白背景上白色不可读 |
| 位置 | 画面下方 1/5，居中 |
| 出现方式 | 逐句淡入，不跳不弹 |
| 大小 | 偏小 |
