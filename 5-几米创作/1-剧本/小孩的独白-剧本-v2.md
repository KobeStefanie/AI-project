| 项目 | 内容 |

| 时长 | 25-30 秒 |

## 情感弧线

```
嘴硬 → 松下来 → 认了 → 说真话 → 呼。
```

## 独白

### 1.

*一片空旷的草地。小孩坐在草地上，猫在不远处蜷着。天空很大，小孩很小。*

「我没有说过讨厌啊。」
「……没有说过。」

*停顿。猫的耳朵动了一下。风很轻。*

---

### 2.

*小孩把脸埋进膝盖里。远处是地平线。*

「好啦。」
「……是我不好。」

*猫抬起头，看了小孩一眼。*

---

### 3.

*小孩侧头看向远方。猫在身旁另一端，中间隔着一点距离。天空开始转色。*

「都很好。」
「只是有时候……想一个人待一会儿。」

*风吹过草地。猫尾巴尖动了一下。*

---

### 4.

*小孩的手指在草地上画圈。猫的眼睛半闭，耳朵朝向小孩。*

「我也有烦恼的。」
「……很多。」

*猫没有动，耳朵一直朝着小孩的方向。*

---

### 5.

*手停了。天光开始变暗。画面里只有很小的小孩和很大很大的天空。*

「大人……永远都不会错。」
「永远都不会错。」

*第二遍更轻，像在重复一个已经相信了的结论。*

---

### 6.

*光线从金黄转成灰蓝。猫走了一步，离小孩近了一点。*

「是我不乖。」
「都是我的问题。」

*说完没有动。猫也没有动。风停了。*

---

### 7.

*小孩抬起头。猫此刻已经在小孩旁边——不是紧挨，但比刚才近。*

「我长大了。」
「会照顾自己的。」

*说第二句的时候很认真，像在跟自己保证。天边最后一抹暖色。*

---

### 8.

*猫站起来，走过来，尾巴绕了一下小孩的脚踝。*

「不要担心。」
「我只是……没有说。」

*全片最轻的一句。说完就散了。天空是一片安静的灰蓝。*

## 尾帧

黑底 + 白色细字手写体 + 右下角极小灰色猫剪影。

**「我只是没有说。」**

静音。1 秒定格。

## 声音设计

| 参数 | 设计 |
|------|------|
| 配乐 | 极简钢琴，单音为主，八音盒质感 |
| 音量 | 低于独白，留白处微微浮上来 |
| 关键点 | 第 5 段「永远都不会错」钢琴一个低音；第 8 段猫走近时钢琴停半拍，然后一个音 |
| 尾帧 | 静音 |

---

## 小孩配音

| 参数 | 要求 |
|------|------|
| 声音 | 中性童声，6-9 岁 |
| 语速 | 偏慢，有停顿，不是背台词 |
| 感觉 | 不是在「配音」，是在「不小心被录到了」 |

## 与旧版（v1）的关键差异

| 维度 | v1 | v2 |
|------|-----|-----|
| 独白对象 | 大人（"你们"） | 猫 / 自己 / 空气 |
| "你们"出现次数 | 7+ 次 | **0 次** |
| 情绪底色 | 辩解、反击、控诉 | 袒露、接纳、呼 |
| 猫的出场 | 仅 2/4 场景 | **全程在场** |
| 尾帧字幕 | 「我只是没有告诉你们而已」 | **「我只是没有说」** |
| 观众看完 | 心疼小孩，对大人有情绪 | 想起自己小时候，安静呼一口气 |
