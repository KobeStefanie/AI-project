# 小孩的独白 — 分镜脚本

> 独立短视频 | 20-30 秒 | 几米水彩风格
> 8 段小孩独白，4 个场景，灰色猫陪伴

---

## 全片时间线

```
场景 A — 房门口（分镜1-2）  0.0s → 6.5s   深蓝 + 门缝暖黄
场景 B — 窗台（分镜3-4）    6.5s → 14.0s  灰蓝 + 月光白
场景 C — 餐桌下（分镜5-6）  14.0s → 20.0s 暖木 + 阴影蓝
场景 D — 走出来（分镜7-8）  20.0s → 27.0s 暖灰 + 晨光金
尾帧                        27.0s → 28.0s 黑底
```

---

## 场景 A · 房门口

**时长**：0.0s — 6.5s（约 6.5 秒）
**色调**：深蓝暗部 + 门缝透出暖黄光
**画面描述**：小孩站在自己房门口，背对镜头，小小身影。门开了一条缝，门缝外面是亮着灯的客厅——但客厅是模糊的，像另一个世界。小孩的手还放在门把手上，没有完全推开，也没有关。灰色猫不在这个场景里。

**AI 图片提示词**：
> 几米绘本风格，水彩湿画法，一个小孩的背面站在半开的房门口，门缝透出暖黄色灯光，房间内是深蓝色暗调，小孩身影很小，大面积留白，画面安静，不悲伤，东方线条感，1024×1536

| 时间 | 字幕/旁白 | 备注 |
|------|----------|------|
| 0.0-1.0s | [画面淡入，无声] | 只出画面，给观众 1 秒进入 |
| 1.0-2.5s | 「我也没有说过我讨厌你们呀！」 | 语气：认真澄清，不是吵架 |
| 2.5-3.0s | [微小停顿] | 门把手还握着，没松 |
| 3.0-4.0s | 「我又没说我不爱这个家」 | 语气：声音小了一点，补充说明 |
| 4.0-5.5s | 「好啦好啦，只有我不好可以了吧！」 | 语气：赌气，但尾音上扬，不是真的委屈 |
| 5.5-6.5s | [停顿 + 画面过渡] | 小孩松开门把手，画面淡出过渡 |

## 场景 B · 窗台

**时长**：6.5s — 14.0s（约 7.5 秒）
**色调**：灰蓝夜色 + 月光白 + 远处街灯暖黄晕染
**画面描述**：小孩坐在窗台上，侧身靠着窗框，光脚，膝盖蜷起来。窗外是夜晚的城市——模糊的水彩楼影和几颗散落的灯光。灰色猫在窗台的另一端，不是紧挨着小孩，中间隔了一个人的距离。猫趴着，尾巴垂下来，眼睛半闭，耳朵朝向小孩。

**AI 图片提示词**：
> 几米绘本风格，水彩湿画法，小孩侧身坐在窗台上靠着窗框，窗外是模糊的夜晚城市水彩楼影，月光白色调，远处暖黄街灯晕染，一只灰色短毛猫安静地趴在窗台另一端，和小孩之间隔着一段距离，大面积留白，暗但不冷，1024×1536

| 时间 | 字幕/旁白 | 备注 |
|------|----------|------|
| 6.5-8.0s | 「这里全都很好啦！」 | 语气：像在回答一个没人问的问题 |
| 8.0-8.5s | [微小停顿] | 窗外有光闪了一下 |
| 8.5-10.0s | 「只是有时候，我也想要一个人静一静嘛」 | 语气：尾音「嘛」拖一点点，撒娇不是求饶 |
| 10.0-10.5s | [停顿] | 猫的尾巴尖动了一下 |
| 10.5-12.0s | 「又不是只有你们有烦恼」 | 语气：平等宣告，不卑不亢 |
| 12.0-13.0s | 「其实我也有很多烦恼」 | 语气：轻下来，说给自己听一样 |
| 13.0-14.0s | [停顿 + 画面过渡] | 小孩低头看自己的脚，画面淡出 |

---

## 场景 C · 餐桌下

**时长**：14.0s — 20.0s（约 6 秒）
**色调**：暖黄色桌布透光 + 下方阴影蓝灰
**画面描述**：小孩蹲在餐桌下面，抱着膝盖。桌布垂下来，形成一个半封闭的小空间——像帐篷、像堡垒。桌布外面透进来暖黄的灯光，但桌下是安静的蓝灰阴影。能看到小孩的光脚和几根桌腿。猫不在这个场景里——这是小孩一个人的小堡垒。

**AI 图片提示词**：
> 几米绘本风格，水彩湿画法，小孩蹲在餐桌下面抱着膝盖，白色桌布垂下来形成半封闭的小空间，桌布外透进暖黄色灯光，桌下是安静的蓝灰色阴影，几根木质桌腿像柱子，光脚踩在地板上，画面像一个秘密堡垒，安全安静，1024×1536

| 时间 | 字幕/旁白 | 备注 |
|------|----------|------|
| 14.0-15.5s | 「对啦，反正大人永远都不会错——永远都不会错」 | 语气：第一遍正常音量，第二遍「永远都不会错」声音变小，像在复读给自己听 |
| 15.5-16.5s | [停顿] | 桌布微微动了一下 |
| 16.5-18.0s | 「全都是我们小孩不乖啦」 | 语气：语气轻快得有点假 |
| 18.0-19.0s | 「对啦，全是我自己的问题」 | 语气：说完低下头，下巴搁在膝盖上 |
| 19.0-20.0s | [停顿 + 画面过渡] | 桌布透的光晃了一下，画面淡出 |

## 场景 D · 走出来

**时长**：20.0s — 27.0s（约 7 秒）
**色调**：暖灰 + 晨光淡金
**画面描述**：小孩从角落站起来，不是从桌下——而是从一个更开阔的角落（靠墙的矮柜旁边？床和墙之间的缝隙？），逆光中小小的剪影。灰色猫从旁边走过来，走到小孩脚边，停下来，尾巴竖起来绕了一下小孩的脚踝。小孩低头看猫。整个画面最后小孩和猫都没有在"躲"——他们只是站在房间的一角，但光是暖的。

**AI 图片提示词**：
> 几米绘本风格，水彩湿画法，小孩的剪影站在房间的角落，逆光中淡金色的晨光从窗户照进来，小孩刚从蹲着的姿势站起来，一只灰色短毛猫走到小孩脚边，尾巴竖着绕了一下小孩的脚踝，画面温暖安静，大面积留白，几米式的温柔收尾，1024×1536

| 时间 | 字幕/旁白 | 备注 |
|------|----------|------|
| 20.0-21.5s | 「我已经长大了，会自己照顾自己的」 | 语气：认真的宣告，但声音不大 |
| 21.5-22.5s | [停顿] | 猫走过来，出现在画面里 |
| 22.5-24.0s | 「拜托喔，你们不要那么爱担心好不好」 | 语气：不是生气，是温柔的抱怨 |
| 24.0-25.5s | 「我只是没有告诉你们而已」 | 语气：全片最轻的一句，说出口就散了 |
| 25.5-27.0s | [无声，画面停留] | 猫的尾巴绕小孩脚踝，小孩低头看猫 |
| 27.0-28.0s | [尾帧淡入] | 黑底 + 白色细字「我只是没有告诉你们而已」 |

---

## 尾帧

**时长**：1 秒（定格）
**画面**：黑底 + 白色细字手写体 + 右下角灰色猫剪影极小
**字幕**：「我只是没有告诉你们而已」
**无音乐，静音**（只有这一帧是安静的，和前面形成落点）

---

## 声音设计

| 参数 | 设计 |
|------|------|
| 配乐 | 极简钢琴独奏，单音为主，少和弦 |
| 音量 | 偏低，不盖独白，画面留白处微微浮上来 |
| 环境音 | 无。不要加环境音，保持干净 |
| 关键音点 | 场景 B 过渡时（14s）钢琴一个低音；场景 D 猫出现时（22s）钢琴停半拍 |

**配乐情绪曲线**：
```
场景 A：八音盒式单音，3-4 个音循环，轻
场景 B：钢琴进入，缓慢下行音阶，留白处琴声浮现
场景 C：钢琴最弱，几乎只有独白
场景 D：钢琴回来，比之前更暖，最后两个音落在尾帧之前
尾帧：静音
```

## 小孩配音提示

| 参数 | 要求 |
|------|------|
| 声音类型 | 中性童声，不分男女，6-9 岁 |
| 语速 | 偏快但不急促，像小孩在自言自语 |
| 语调 | 不上扬不表演，不装可爱，自然说话 |
| 关键情绪 | 分镜1 认真 → 分镜2 赌气 → 分镜3 无奈 → 分镜4 平等 → 分镜5 讽刺 → 分镜6 自嘲 → 分镜7 认真 → 分镜8 温柔 |
| 整体感觉 | 不是在"配音"，是在"不小心被听到了" |

## 剪辑指导

### 画面切换

| 过渡 | 方式 | 时长 |
|------|------|------|
| A → B | 淡出淡入 | ~1s |
| B → C | 淡出淡入 | ~1s |
| C → D | 淡出淡入 | ~1s |
| D → 尾帧 | 硬切 | 0s |

### 字幕样式

| 参数 | 要求 |
|------|------|
| 字体 | 思源黑体 Light 或类似细体无衬线 |
| 颜色 | 白色，略微降低不透明度（~90%） |
| 位置 | 画面下方 1/5，居中 |
| 出现方式 | 逐字或逐句淡入，不跳，不弹 |
| 大小 | 偏小，不要撑满画面宽度 |

### 画面节奏

- 每个场景的画面不是静止的——极慢的 Ken Burns（缓慢推进或横移，幅度极小）
- 场景切换处留够黑屏过渡帧，不要硬跳
- 尾帧的「我只是没有告诉你们而已」和前面场景 D 的最后一句独白是同一句话——场景 D 的独白说完后，画面继续停留 1.5 秒，然后才硬切到尾帧黑底重写一遍这句话。两遍「我只是没有告诉你们而已」，第一遍是说的，第二遍是写的。
