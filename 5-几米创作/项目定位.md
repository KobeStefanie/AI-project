# 几米创作 — 项目定位

> 项目级配置文件。所有子项目共享此定位，AI 在步骤 1（写故事或剧本）开始前必须读取本文件。

## 1. 风格

**几米式水彩绘本风为主基调（80%），按情绪状态切换视觉配方。**

| 元素 | 说明 |
|------|------|
| **主色调** | 灰蓝 + 暖黄固定调色板，暖灰暮光质感 |
| **构图** | 大场景小人物、背影/侧影、大量留白、框中之框（窗框/门框） |
| **媒材感** | 水彩 + 彩铅、手工墨线、纸纹可见、湿画法晕染 |
| **情绪温度** | 5-15°C，不冷到绝望也不暖到推销 |
| **人物** | 无五官或小点眼睛、头身比偏 Q 不幼稚、背对/侧对镜头为主 |

**视觉配方切换（五位画家体系）：**
- 日常陪伴 → 几米（灰蓝+暖黄、空旷、可呼吸）
- 极端压抑 → 陈志勇（灰暗工业色调、单点红色希望）
- 沉默/等待 → 寂地（昏暗底+光晕、无五官黑洞眼）
- 独处/自然 → Tove Jansson（淡雅北欧调、大海岛屿森林）
- 释放/轻盈 → Rébecca Dautremer（暖金逆光、漂浮感、柔焦）

## 2. 核心主题

**不给答案、不评判、不规劝——创造一个让人透口气的空间。**

核心信念：每个人内心都是向阳的，只是被压抑和损耗了。给他空间，他自己会恢复。

内容哲学：
- 画面是镜子，文字是镜框——不是你去看见他，是他自己看见自己
- 路过 → 停下来 → 自己发现 → 呼
- 「这条内容，是在给观众一个透口气的空间，还是在给他一个需要回应的东西？」

## 3. 定位

**「内容公司」——用几米视觉语言构建的心理安全空间。**

账号名：**角落先生**
简介：**一个角落，门开着。在听。**

平台差异化：唯一同时占有「高审美视觉 + 专业心理学 + 咨询师身份 + 不给答案不规劝」的位置。

平台策略：
- 小红书 ⭐⭐⭐⭐⭐ 主战场
- 抖音 ⭐⭐⭐⭐ 辅助分发
- 视频号 ⭐⭐⭐ 品牌背书

## 4. 目标受众

**25-35 岁都市女性为主。**

特征：
- 有情绪困扰但不想被"治疗"
- 渴望被理解但不想被同情
- 需要一个不被打扰的角落
- 对审美有要求、能感受文字的力量
