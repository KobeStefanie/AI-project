## 二、全局视听设定

### 2.1 画面风格

| 参数 | 设定 |
|------|------|
| 色彩基调 | 暖米白主调，灰蓝点缀（猫），亮橘点缀（小孩T恤），不调色不滤镜 |
| 颗粒感 | 保留水彩纸纹，不加胶片颗粒 |
| 对比度 | 低对比，柔和——不要高反差 |
| 画面运动 | 每镜极慢 Ken Burns，幅度极小（全程 ≤3% 缩放/位移） |
| 运动曲线 | ease-in-out，无线性匀速——像呼吸 |

### 2.2 旁白风格

| 参数 | 设定 |
|------|------|
| 声线 | 中性童声，6-9 岁，不分男女 |
| 语速 | 140-160 字/分钟，偏慢，不急促 |
| 语调 | 不上扬不表演，不装可爱，自然说话 |
| 录音感觉 | 近距离轻声，像坐在旁边自言自语，不是对着麦克风念稿 |
| 响度 | -16 LUFS（整合响度），峰值 -6 dBTP |

### 2.3 配乐风格

| 参数 | 设定 |
|------|------|
| 乐器 | 钢琴独奏，极简，单音/双音为主，少和弦 |
| 音色 | 柔音踏板（soft pedal），毛毡钢琴更佳 |
| 速度 | ♩ = 60-72 bpm，缓慢 |
| 调性 | C 大调或 A 小调，中性，不悲不喜 |
| 混音 | 干声为主，少量大厅混响（reverb ~20% wet） |
| 响度 | -24 LUFS，低于旁白 8 LUFS |

### 2.4 音效

| 参数 | 设定 |
|------|------|
| 环境音 | **无**。不加任何环境音、白噪音、氛围底噪 |
| 关键音效 | 仅 1 处——分镜 8 猫蹭腿时，极轻一声「呼噜」（cat purr），-30 dB，若有若无 |

## 三、分镜制作明细

> **景别说明**：全片景别统一为**远景（Wide Shot / Extreme Wide Shot）**，人物占画面高度 15-20%。不做景别切换——保持「远远看着」的呼吸距离。
