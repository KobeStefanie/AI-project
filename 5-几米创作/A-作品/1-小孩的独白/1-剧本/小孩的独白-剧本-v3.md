> 米白色背景 + 大量留白 + 小孩与猫
> 情绪纯靠人物姿态和猫的形态传递

---

## 基本信息

| 项目 | 内容 |
|------|------|
| 作品名 | 小孩的独白 |
| 类型 | 独立短视频 |
| 时长 | 25-30 秒 |
| 角色 | 小孩（梨形脸，小点眼睛或无五官）+ 灰色猫（全程在场） |
| 背景 | 米白色，极简。可能只有一条极淡的线暗示落脚点。大量留白 |
| 风格 | 几米水彩，大场景小人物，80%+ 留白 |

---

## 情感弧线

```
嘴硬 → 松下来 → 认了 → 说真话 → 猫蹭过来 → 抱住 → 呼。
```

---

## 猫的距离弧线

```
分镜 1-4：不远不近，安静在场
分镜 5-6：走近一步
分镜 7：很近，不挨
分镜 8：主动蹭 → 露肚皮 → 被抱起
```

## 独白

### 1.

*米白色背景。小孩坐着，抱着膝盖。猫在不远处蜷成一团。*

「我没有说过讨厌啊。」
「……没有说过。」

*猫的耳朵动了一下。*

---

### 2.

*小孩把脸埋进膝盖里。猫抬起头，看向小孩。*

「好啦。」
「……是我不好。」

---

### 3.

*小孩侧头，望向画面外的空白处。猫在另一端，中间隔着距离。猫尾巴尖微微动了一下。*

「都很好。」
「只是有时候……想一个人待一会儿。」

---

### 4.

*小孩的手指在脚下画圈。猫眼睛半闭，耳朵朝向小孩的方向。*

「我也有烦恼的。」
「……很多。」

---

### 5.

*小孩的手停了。一动不动。画面里小孩很小，留白很大。猫睁开眼睛。*

「大人……永远都不会错。」
「永远都不会错。」

*第二遍更轻。猫走了一步，离小孩近了一点。*

---

### 6.

*小孩下巴搁在膝盖上。猫又近了一步，但还是隔着一点点距离。*

「是我不乖。」
「都是我的问题。」

---

### 7.

*小孩抬起头。猫此刻已经在小孩旁边——不挨着，但伸手就能够到。*

「我长大了。」
「会照顾自己的。」

*说第二句的时候很认真，像在跟自己保证。*

---

### 8.

*猫站起来，走到小孩身边，用头蹭了蹭小孩的腿。然后整个身体靠过去，蜷在小孩脚边，翻了个身露出肚皮。*

*小孩低头看猫，愣了一下。伸手摸了摸猫的头。*

「不要担心。」
「我只是……没有说。」

*全片最轻的一句。小孩把猫抱起来，脸埋在猫的背上。画面停留。*

## 尾帧

米白色背景淡出 → 黑底。
白色细字手写体：**「我只是没有说。」**
右下角，极小灰色猫剪影。
静音。1 秒定格。

---

## 声音设计

| 参数 | 设计 |
|------|------|
| 配乐 | 极简钢琴，单音为主，八音盒质感 |
| 音量 | 低于独白，留白处微微浮上来 |
| 关键点 | 第 5 段「永远都不会错」钢琴一个低音；第 8 段猫蹭过来时钢琴停半拍，然后一个暖音 |
| 尾帧 | 静音 |

---

## 小孩配音

| 参数 | 要求 |
|------|------|
| 声音 | 中性童声，6-9 岁 |
| 语速 | 偏慢，有停顿，不是背台词 |
| 感觉 | 不是在「配音」，是在「不小心被录到了」 |
