# 躲进世界的角落 — 系列规划

> 灵感来源：几米《躲进世界的角落》
> 核心视角：一个小孩的独白，"我心里有一个小孩，她每一次躲进角落，都只是暂时的缺席"

---

## 系列定位

| 决策项 | 结论 |
|--------|------|
| 系列名称 | **「躲进世界的角落」** |
| 叙述视角 | 小孩第一人称（不露面，只出现小身影+独白） |
| 系列语调 | 安静的好奇，不悲伤不卖萌，"大人们不知道的事" |
| 时长 | **15-20 秒**（统一，三平台共用） |
| 调色 | 暗但不冷，阴影里有暖光，角落有秘密 |
| 发布节奏 | 周更 2 条 |
| 平台 | 抖音 / 小红书 / 视频号 |

## 与旧版的根本区别

| 维度 | 旧「城市里的人」 | 新「躲进世界的角落」 |
|------|-----------------|---------------------|
| 叙述者 | 成人独白 | **小孩的口吻** |
| 情绪底色 | 沉重的孤独 | **安静的观察** |
| 夜晚的意义 | 恐惧醒着 | **世界终于安静了** |
| 猫的角色 | 偶尔出现的安慰 | **唯一懂我的伙伴，每集都在** |
| 字幕风格 | 格言式、深沉 | **小孩话，简单的真相** |
| 结尾 | 温暖微光 | **不打烊的发现——原来角落是暖的** |

## 7 集序列

### 001 — 大人都睡着了

| 维度 | 内容 |
|------|------|
| 小孩独白 | "大人都睡了以后，房子才开始呼吸。" |
| 情绪线 | 白天吵 → 深夜静 → 发现秘密：冰箱在嗡嗡叫，水管在咕噜，窗帘自己在动 |
| 画面 | 小人从卧室门缝偷看黑暗客厅 → 光脚踩过地板 → 爬上窗台 |
| 结尾 | 猫已经在窗台上等着了，好像一直都知道她要来 |
| 色调 | 深蓝 + 月光白 |

### 002 — 猫知道我在哪

| 维度 | 内容 |
|------|------|
| 小孩独白 | "每次我躲起来，只有猫找得到我。大人永远找不到。" |
| 情绪线 | 躲藏的游戏 → 大人经过但看不见 → 猫轻轻走来，坐在藏身处外面 |
| 画面 | 桌布下面露出一双小脚 → 大人的腿走来走去 → 猫走过来，尾巴勾了一下桌布 |
| 结尾 | 小孩从桌布下伸出手指，和猫的爪子碰了一下 |
| 色调 | 暖木色 + 桌布透光 |

### 003 — 大人的嘴巴好忙

| 维度 | 内容 |
|------|------|
| 小孩独白 | "大人的嘴巴一直动一直动，但说的话好少。" |
| 情绪线 | 饭桌上大人聊个没完 → 小孩在下面看他们的脚 → 发现大人抖腿、蹭地、脚尖互相碰 |
| 画面 | 餐桌下方视角，桌面上全是盘子碗筷的阴影，桌面下是各种腿和脚的小动作 |
| 结尾 | 猫也躲在桌子下面，和小孩一起看这场"脚戏" |
| 色调 | 暖黄桌面光 + 阴影下的蓝灰 |

### 004 — 下雨的时候

| 维度 | 内容 |
|------|------|
| 小孩独白 | "下雨的时候，整个城市都变软了。" |
| 情绪线 | 窗外的雨声 → 街灯的光晕被雨打散 → 所有硬的东西都模糊了 |
| 画面 | 小孩趴在窗边 → 雨滴在玻璃上赛跑 → 窗外城市一片水彩晕开的朦胧 |
| 结尾 | 猫跳到窗台另一边，用爪子拍了一下玻璃上的雨滴 |
| 色调 | 灰蓝 + 街灯暖黄晕染 |

### 005 — 大人肚子里都住着人

| 维度 | 内容 |
|------|------|
| 小孩独白 | "每个大人肚子里都住着另一个小小的人。我看得见，他们自己看不见。" |
| 情绪线 | 街上看路人 → 每个人身边飘着不同形状的小影子 → 有的很淡快消失了 |
| 画面 | 小孩和猫在街边长椅上 → 路人走过，每个人旁边有一个透明的小人影子（超现实） |
| 结尾 | 一个穿着西装急急忙忙的大叔走过，他的小影子停下来朝小孩挥了挥手 |
| 色调 | 灰蓝街景 + 暖黄透明小人影 |

### 006 — 可以不说话

| 维度 | 内容 |
|------|------|
| 小孩独白 | "我和猫可以一整个下午坐在楼梯上，什么都不说。" |
| 情绪线 | 安静的午后 → 阳光一格一格爬楼梯 → 猫换了个姿势 → 又换了一个 |
| 画面 | 楼梯间俯角 → 小孩和猫各占一级台阶 → 光影缓慢移动 |
| 结尾 | 猫突然站起来，走过来蹭了一下小孩，又回去了 |
| 色调 | 暖灰 + 午后斜阳金 |

### 007 — 角落不是躲的地方

| 维度 | 内容 |
|------|------|
| 小孩独白 | "你以为我躲起来了。我只是在等世界安静下来，再重新走回去。" |
| 情绪线 | 回忆前面 6 集的各种角落 → 每一个角落现在都有微弱的光 |
| 画面 | 蒙太奇：窗台/桌下/楼梯/窗边/街边长椅 → 最后回到第一个角落，但这次天亮了 |
| 结尾 | 小孩站起来，猫跟在后面，一起走回亮起来的房间 |
| 色调 | 系列所有色调的融合 → 最终转为淡暖晨光 |

## 系列视觉规则

| 规范 | 标准 |
|------|------|
| 小孩形象 | 永远不露正脸（背影/侧面/剪影/只露手脚），几米式中性 |
| 猫 | 灰色短毛猫，安静，不卖萌，每集换位置但一集不漏 |
| 色彩基调 | 暗但不冷——阴影里永远有一点暖光（月光/街灯/桌灯/雨反光） |
| 每集画面 | 4-5 张，15-20 秒，3-5 秒/张 |
| 字幕 | 细体无衬线，画面下方 1/4，**小孩手写体感觉**（思源黑体 Light 或类似） |
| 配乐 | 钢琴独奏 > 八音盒 > 轻柔弦乐，音量偏低 |
| 尾帧 | 黑底 + 白色细字「躲进世界的角落」+ 猫剪影 |
