## 转场设置

| 切点 | 转场 | 时长 | 说明 |
|------|------|------|------|
| 场1 → 场2 | **硬切** | 0s | 直接跳，产生"进入室内"的感觉 |
| 场2 → 场3 | **无转场** | — | 用"缓推近"动效代替切 |
| 场3 → 场4 | **硬切** | 0s | 视角切换（室内→窗外） |
| 场4 → 场5 | **叠化** | 1s | 柔和的过渡，表现时间流走 |

---

## 动效设置（剪映操作）

剪映中选中图片 → 「动画」→ 「组合动画」或手动关键帧：

| 场景 | 动效方式 | 操作 |
|------|----------|------|
| scene-01 | 极缓下移 | 起始位置略上 → 结束位置略下，持续整段 |
| scene-02 | 静止 | 不加任何动效，纯静帧 |
| scene-03 | 极缓推近 | 从 100% → 105% 缩放，焦点往人物方向 |
| scene-04 | 静止 | 不加动效（画面的天色是图里画好的渐变） |
| scene-05 | 极缓拉远 | 从 105% → 100% 缩放 |

> ⚠️ 所有动效必须**极慢**。几米风的关键是呼吸感，动太快就破了。

## 字幕设置

| 参数 | 值 |
|------|-----|
| 字体 | 思源黑体 Light / 苹方 细体 |
| 颜色 | 白色，透明度 80% |
| 大小 | 中等偏小（约占画面高度 3-4%） |
| 位置 | 画面下方 1/4 处，水平居中 |
| 动画 | **淡入**（0.5s），不要跳字、不弹出 |
| 持续时间 | 字幕所在场景的整段时长 |

只两句字幕：
- 场2："整座城市都睡了"
- 场3："你和你的恐惧还醒着"

## 配乐建议

剪映音频 → 音乐 → 搜索以下风格：

| 推荐 | 类型 | 感觉 |
|------|------|------|
| 首选 | 钢琴独奏 | Yiruma 类，缓慢下行再上行 |
| 备选 | 大提琴 | 深沉但不沉重 |
| 备选 | 环境音 | 轻微的夜风声 + 远处偶尔的车声 |

搜索关键词建议：`钢琴 安静 治愈` `深夜 钢琴` `轻音乐 抒情`

音量：**-12dB 到 -14dB**，不要抢画面。
