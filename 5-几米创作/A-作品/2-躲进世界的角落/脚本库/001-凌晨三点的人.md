## 分镜

### 场景 1 — 城市沉睡 [深]

| 维度 | 内容 |
|------|------|
| 画面 | 俯瞰城市夜景。万家灯火已灭，只剩零星几扇窗亮着。深蓝色夜空压在城市轮廓上方。画面大部分是暗的，灯火是极小极疏的点。 |
| 构图 | 大俯角，一点透视，地平线在画面上方 1/3 处 |
| 几种米风要点 | 小人物大背景（城市是小人物，人是更小的点）、深蓝水彩晕染 |
| 字幕 | （无） |
| 时长 | 抖音 3s / 小红书 5s / 视频号 4s |
| 动效 | 极缓慢向下平移 |

---

### 场景 2 — 窗台上的人 [深]

| 维度 | 内容 |
|------|------|
| 画面 | 切换到室内。一个极小的人影坐在窗台上，背对镜头，面向窗外。房间内部几乎全暗，只有窗外的城市微光照出他/她的轮廓。窗框形成一个"框中之框"，把小人更压得更小。 |
| 构图 | 平视，人物占画面约 8%，窗框占据大部分 |
| 几种米风要点 | 人物背面/无五官（几米式中性处理）、画面大部分留黑 |
| 字幕 | "整座城市都睡了" |
| 时长 | 抖音 3s / 小红书 5s / 视频号 4s |
| 动效 | 静止，字淡入 |

---

### 场景 3 — 恐惧也在 [深]

| 维度 | 内容 |
|------|------|
| 画面 | 镜头推近一点。小人依然坐在窗台。窗外很远的夜空中，隐约飘着一团深色的"东西"——说不清是云、是雾、还是什么形状。它不狰狞，但存在。小人看着它。 |
| 构图 | 中景，小人在画面右下 1/4，那团暗影在左上远空 |
| 几种米风要点 | 超现实元素（不确定形状的存在物 = 恐惧的视觉化），灰蓝调配色 |
| 字幕 | "你和你的恐惧还醒着" |
| 时长 | 抖音 4s / 小红书 6s / 视频号 5s |
| 动效 | 极缓慢推近（往小人方向），暗影微微浮动 |

---

### 场景 4 — 第一缕光 [转折开始]

| 维度 | 内容 |
|------|------|
| 画面 | 窗外的远处天际线，第一缕灰蓝色的晨光透出来。不是耀眼的金光，是很淡、很不确定的蓝灰色微光。城市轮廓开始隐约可见。暗影在变淡。 |
| 构图 | 窗外视角，天际线在画面中央偏下 |
| 几种米风要点 | 几米中期淡彩柔和调、大面积水彩晕染的天空过渡 |
| 字幕 | （无） |
| 时长 | 抖音 3s / 小红书 5s / 视频号 4s |
| 动效 | 天空色彩极缓慢从深蓝向灰蓝渐变 |

---

### 场景 5 — 猫 [暖]

| 维度 | 内容 |
|------|------|
| 画面 | 切回室内。小人依然坐在窗台上，只是姿势从双臂抱膝——松开了。不知道什么时候，一只猫蹲在了他/她身旁的窗沿上。猫没有看小人，猫看着窗外，尾巴安静地垂着。窗外天光比刚才更亮一点，仍是清晨的灰蓝色。猫的轮廓被微光勾出一圈很淡的暖灰边。 |
| 构图 | 侧面/斜侧面，小人和猫分别占据窗台两端，中间留白 |
| 几种米风要点 | 猫=几米经典符号（沉默的陪伴）、画面大量留白、不安排猫看人（更真实）、暖来自天光而不是物件 |
| 字幕 | （无） |
| 时长 | 抖音 5s / 小红书 7s / 视频号 7s |
| 动效 | 猫尾巴极轻微地晃了一下，画面极缓慢拉远 |

## 情绪弧线

```
深 ────→ 最深 ────→ 微光 ────→ 暖
场1        场2+3       场4        场5
城市睡了    恐惧醒着    天亮了     猫来了
```

## 剪辑备忘

- 场 1 → 场 2：硬切
- 场 2 → 场 3：极慢推近
- 场 3 → 场 4：硬切（视角转换）
- 场 4 → 场 5：叠化（1s），表现时间的轻柔过渡
- 字幕出现在画面下方 1/4，字体：细体无衬线（如思源黑体 Light），白色 80% 透明度
- 全程无旁白，只有背景钢琴
