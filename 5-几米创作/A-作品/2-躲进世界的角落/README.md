# 躲进世界的角落

> 灵感来源：几米《躲进世界的角落》

| 项目 | 内容 |

## 创作原则

- 小孩的视角，小孩的语言
- 不解释、不诊断、不讲道理
- 暗但不冷，角落里有光
- 猫知道一切，但猫不说
