# 励志图片生成器

一个简单的励志文字图片生成工具，可以将文字内容转换为黑底白字的社交媒体风格图片。

## 功能特点

- 黑色背景 + 白色文字的简约设计
- 参考流行的励志文案排版风格
- 支持自定义文字内容
- 一键下载为 PNG 格式（1080x1080）
- 无需安装，浏览器直接使用

## 使用方法

1. 打开 `src/index.html`
2. 页面会自动生成预设文字的图片
3. 点击"下载图片"按钮保存

## 技术栈

- HTML5 Canvas
- Tailwind CSS
- 纯前端实现，无需后端

## 示例内容

默认生成的图片包含以下内容结构：
- 顶部：To everyone:
- 中间：励志文字内容
- 底部：收到回复：OK
