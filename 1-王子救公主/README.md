# 1-王子救公主

一个单文件网页小游戏：勇者打怪升级，最终化身白马王子，抱着身披婚纱的公主奔向远方。

## 目录结构

```
1-王子救公主/
├── README.md
└── src/
    └── 1-王子救公主.html    # 游戏主文件（含 HTML / CSS / JS）
```

## 运行方式

双击 `src/1-王子救公主.html`，用任意现代浏览器打开即可，无需安装依赖。

## 玩法

- **⚔️ 攻击**：基础输出
- **🔥 火球术**：消耗 10 MP，约 2.2 倍伤害
- **🧪 喝药水**：20 金币，回 50 HP + 10 MP
- **🌹 送公主礼物**：50 金币，提升好感度

依次击败：史莱姆 → 哥布林 → 骷髅兵 → 兽人 → 黑暗骑士 → 巨龙 → 魔王。
